## 约定
- 颜色规则：涨红跌绿（A 股习惯）
- 语言：简体中文
- 内容格式：Markdown 为主
- 目录扩展：按需渐进，不预设空目录
- **审查规则（强制）**：每次 commit + push 前，必须先启动审查 Agent 对修改内容做准确性/完整性/一致性/时效性四重检查
- **业务线核验铁律**：凡写公司「业务构成 / 营收拆分」，必须用 neodata-financial-search 的「主营构成」交叉验证，**不得只信旧快照或训练数据**。多业务公司（华工=光电+激光+传感器、剑桥=电信宽带+光模块+无线）最易漏写非主业板块或高估单一业务占比；子集营收（如光模块）不得大于其所属母类（如光电器件系列）。
- **26Q1 数据铁律**：neodata 结构化利润表接口仅覆盖至 2025Q4，多数 A 股无 26Q1 数据。**凡 26Q1 单季营收/净利超过 2025 全年同口径数（尤其单季归母 > 全年归母），必为单位/口径错误（常见 10× 量级），必须删除并标「neodata 未收录」，严禁保留疑似错误数**。已踩坑：存储芯片板块佰维/江波龙/德明利/香农芯创 26Q1 归母远超全年，2026-07-15 已修。
