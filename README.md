# Stock — 炒股基本面学习项目

从零构建的跨市场投资学习体系，覆盖 **A 股 / 港股 / 美股**，聚焦基本面分析。

## 学习框架

| 模块 | 内容 | 市场覆盖 |
|------|------|----------|
| 板块产业链 | 上游 → 中游 → 下游全景梳理 | A股 / 港股 / 美股 |
| 板块细分 | 细分赛道、竞争格局、龙头对比 | A股 / 港股 / 美股 |
| 公司概况 | 主营业务、护城河、股东结构 | A股 / 港股 / 美股 |
| 财务分析 | 三张表解读、关键财务指标 | A股 / 港股 / 美股 |
| 资金流向 | 北向资金、主力动向、机构持仓 | 跨市场 |
| 宏观数据 | GDP、CPI、利率、政策跟踪 | 全球 |
| 技术分析 | TradingView 图表分析记录 | 跨市场 |
| 投资策略 | 投资体系、交易纪律、仓位管理 | 通用 |

## 使用方式

1. 按模块浏览对应目录下的 Markdown 文件
2. 每个行业 / 公司一个独立文件，便于维护
3. 持续更新，随着学习进度不断补充

## 技术栈

- 内容格式：Markdown
- 数据来源：ardot-design-assistant 技能 + TradingView
- 版本管理：Git + GitHub
