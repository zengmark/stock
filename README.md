# Stock — 炒股基本面学习项目

从零构建的跨市场投资学习体系，覆盖 **A 股 / 港股 / 美股**，聚焦基本面分析。

## 学习框架

| 模块 | 内容 | 目录格式 |
|------|------|----------|
| 板块研究 | 产业链 + 细分赛道，合并研究 | `行业 → 市场 → 文件` |

> 后续模块随学习进度添加，不预设空目录。

## 使用方式

1. 按模块浏览对应目录下的 Markdown 文件
2. 每个行业 / 公司一个独立文件，便于维护
3. 持续更新，随着学习进度不断补充

## 技术栈

- 内容格式：Markdown
- 版本管理：Git + GitHub
