# 02 产业链深度拆解

> 一颗存储芯片从沙子到装机，链条比想象中长。本档把「设计 → 制造 → 封测 → 模组 → 终端」拆开，并标出每个环节 A 股能投什么。

## 2.1 全链条视图

```
设计（IP/电路）
   │
   ▼
晶圆制造（前道，DRAM/NAND 原厂自建产线 or 代工）
   │
   ▼
封装测试（后道，DRAM 颗粒 / NAND 颗粒 / HBM 堆叠）
   │
   ▼
模组（把颗粒 + 主控 + 固件做成 SSD / 内存条 / ePOP）
   │
   ▼
终端（AI 服务器 / PC / 手机 / 汽车 / 工业）
```

> 上游还有**设备**（光刻/刻蚀/薄膜，被美日荷垄断）与**材料**（硅片/靶材/光刻胶）。存储原厂资本开支的 30–40% 流向设备，是半导体设备的隐藏需求。

## 2.2 环节一：设计（IP 与电路）

- **原厂自有设计**：三星 / SK 海力士 / 美光 / 铠侠自研 DRAM/NAND 电路与制程。
- **Fabless 利基设计（A 股重点）**：兆易创新（NOR+DRAM+MCU）、北京君正（车规 DRAM/NOR，ISSI）、东芯（NAND/NOR/DRAM）、普冉（NOR/EEPROM）、聚辰（EEPROM+DDR5 SPD）。这类公司不做最先进制程，但在利基市场有份额与壁垒。

## 2.3 环节二：晶圆制造（前道）

- DRAM/NAND 是**重资产、IDM 模式**——原厂自己建厂（三星/海力士/美光/铠侠），不像逻辑芯片大量依赖台积电代工。
- A 股无纯 DRAM/NAND 晶圆厂；长鑫存储（未上市）是大陆 DRAM 自主希望，兆易等与其有联合定制合作。

## 2.4 环节三：封装测试（后道，AI 关键）

- **HBM 封测是核心瓶颈**：HBM 需 TSV 硅通孔堆叠 + 与 GPU 的 CoWoS 合封，技术难度极高，产能受限于 SK 海力士 / 三星自家产线及台积电 CoWoS。
- **A 股封测标的**：深科技（沛顿科技，存储封测龙头）、太极实业（海太半导体，承接 SK 海力士封测外包）。这是 A 股离「HBM 增量」最近的后道环节。
- 佰维存储等模组厂也在布局晶圆级先进封测（FOMS/CMC），向上游延伸。

## 2.5 环节四：模组（A 股最密集）

- 把原厂颗粒 + 自研主控 + 固件，做成消费级 / 企业级 SSD、内存条、ePOP（端侧 AI 用）。
- **A 股主力**：江波龙（Lexar，NAND 模组龙头）、佰维存储（AI 端侧 + 封测）、德明利（NAND 主控/模组）、香农芯创（SK 海力士授权分销 + 海普自研）。
- 模组厂的弹性来自「存储涨价周期 + 企业级/AI 端侧增量 + 自研主控提升附加值」。

## 2.6 环节五：终端拉动

- **AI 服务器**：HBM（原厂）+ 高容量 DDR + 企业级 SSD（eSSD）——价值量最大的增量。
- **AIPC / AI 手机**：单机 DRAM/NAND 容量提升（如 AI 手机 12→16GB+），拉动利基存储与 NOR。
- **汽车 / 工业**：车规 DRAM/NOR/EEPROM（北京君正、聚辰），智能化提升单车存储用量。

## 2.7 投资落点小结

| 环节 | A股能投 | 弹性来源 |
|------|---------|---------|
| 设计（利基） | 兆易/君正/东芯/普冉/聚辰 | 国产替代 + 车规/AI 端侧 |
| 制造 | （长鑫未上市） | — |
| 封测 | 深科技/太极实业 | HBM 后道增量 |
| 模组 | 江波龙/佰维/德明利/香农 | 涨价周期 + 企业级/端侧 |
| 分销 | 香农芯创 | 原厂紧缺代理红利 |
