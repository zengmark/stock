# 第五章：海外核心公司分析

---

## 5.1 台积电（TSMC / 2330.TW / TSM）

### 为什么一家代工厂成了先进封装的王者？

台积电在先进封装的统治力，**不是因为它封装做得比 OSAT 好，而是因为它把封装和代工绑定了**。逻辑很简单：

1. 芯片在台积电制造（代工）
2. 代工完成后，芯片已经在台积电的厂里了
3. 台积电说「封装也在我这做吧，一站式搞定，良率我负责」
4. 客户没有选择——因为最先进的 CoWoS 只有台积电能做

**这不是市场竞争，是技术垄断。** 台积电造了硅中阶层（2.5D 封装的灵魂），CoWoS 封装依赖硅中阶层，而硅中阶层只有台积电（和三星，但规模和良率差很远）能产。OSAT 日月光、长电没有晶圆厂，造不了硅中阶层，所以做不了最先进的 2.5D 封装。

### 封装业务三大平台

| 平台 | 技术 | 量产年份 | 当前主力客户 | 月产能（2025E） |
|------|------|---------|------------|----------------|
| **CoWoS** | 2.5D 硅中阶层封装 | 2012 | 英伟达（最大）、AMD、博通 | 6-7 万片 |
| **InFO** | 晶圆级扇出封装 | 2016 | 苹果（A 系列处理器） | 12 万片+ |
| **SoIC** | 3D 混合键合 | 2022 | AMD（3D V-Cache）| 小规模（数千片） |

### 封装业务财务估算

| 指标 | 估算 |
|------|------|
| 先进封装营收（2024 年） | 约 180-200 亿美元（含 CoWoS + InFO + SoIC） |
| 封装占台积电总营收比例 | ~25-30%（且仍在快速提升） |
| 封装毛利率 | 高于代工平均毛利率（CoWoS 是「垄断税」定价） |
| 产能利用率 | CoWoS 连续 2 年超 100%（加班加点仍供不应求） |

### 投资关联（对于关注 A 股先进封装的投资者）

台积电本身在美股/台股交易，但它的封装业务直接关联 A 股多个标的：

| 台积电动向 | A 股受益方向 |
|-----------|------------|
| CoWoS 产能紧缺 → 外溢订单 | 日月光、**长电科技**（间接承接外溢） |
| CoWoS 继续扩产 → 需要设备 | **北方华创**（刻蚀/沉积设备） |
| SoIC 量产推进 → 3D 封装成气候 | 掌握 2.5D/3D 技术的 A 股封装厂 |
| 先进封装产能翻倍 → 基板/材料需求暴增 | **深南电路**（ABF 载板）、**雅克科技** |

## 5.2 日月光投控（ASE / 3711.TW）

### 基本面

| 维度 | 信息 |
|------|------|
| **定位** | 全球最大 OSAT（独立封装测试厂），封装市占率 ~18%，全球排名第 2（仅次于台积电） |
| **商业模式** | 纯第三方的来料封装服务——「你不绑定任何代工厂，你来我这封」 |
| **封装平台** | VIPack（集成 Fan-out、2.5D、3D 等全系先进封装技术） |
| **核心客户** | 高通、联发科、苹果（部分）、英特尔（封装外包部分） |

### 台积电 CoWoS 外溢的最大受益者

日月光最核心的投资逻辑是：**它是台积电 CoWoS 产能不足时，客户最优先考虑的「外包补充」选项**。

为什么是日月光而不是长电？
- 日月光有 VIPack 平台，技术能力和台积电在多个封装领域可以做到兼容替代
- 日月光和台积电同在台湾，地缘协同效率最高
- 日月光与英伟达/AMD/高通等大客户有长期稳定的生产关系

但要注意：外溢订单对日月光是「增量」，不是「替代」。台积电 CoWoS 产能缺口越大，日月光的增量越多。

### 投资关联（A 股）

| 关联 | 说明 |
|------|------|
| 日月光扩产 → 设备采购增加 | 利好北方华创等设备商 |
| 日月光承接外溢 → 间接验证封装产业链景气度 | A 股封装公司整体受益于行业高景气 |
| 日月光 VIPack 不断迭代 → 给长电/通富指明了追赶方向 | 技术参照价值 |

## 5.3 三星电子（005930.KS）

### 三星在先进封装中的位置

三星在先进封装上的路径是：**利用 HBM（高带宽内存）的全产业链优势，试图在 2.5D/3D 封装领域打造「HBM+逻辑芯片」的一体化方案**。

| 维度 | 信息 |
|------|------|
| **封装平台** | I-Cube（2.5D，对标 CoWoS）、X-Cube（3D，对标 SoIC） |
| **HBM 优势** | 三星是全球三大 HBM 供应商之一（SK 海力士、三星、美光），自产 HBM + 自封装 |
| **代工能力** | 有三星的晶圆代工（仅次于台积电），可以自产硅中阶层 |
| **外部客户** | 极少（谁会给竞争对手三星代工+封装？） |
| **市占率** | 远低于台积电，封装客户的开放性是其最大瓶颈 |

### 对 A 股的影响

三星封装能力的提升对 A 股封装厂影响不大（三星封装不对外，不构成竞争）。但三星 HBM 扩产会拉动封装设备和材料需求，间接利好 A 股设备/材料标的。

## 5.4 英特尔（INTC）

### 英特尔在先进封装中的位置

英特尔在先进封装上有深厚积累，但问题是：**封装能力服务于 IDM 2.0 战略，而非独立创造利润**。

| 维度 | 信息 |
|------|------|
| **封装平台** | EMIB（嵌入式硅桥 2.5D）、Foveros（3D 堆叠） |
| **技术特色** | EMIB 不用整片中阶层，只在需要互联的地方嵌入一小块硅桥——理论上成本更低 |
| **商用进展** | Meteor Lake（2023）首次大规模商用，但产品市场反响一般 |
| **IDM 2.0 的逻辑** | 英特尔想用先进封装吸引代工客户——「把芯片给我代工，我就用 EMIB/Foveros 帮你封装」 |

### 对 A 股的影响

英特尔先进封装的成败，取决于 IDM 2.0 代工战略能否成功——这不是封装本身能决定的。对 A 股投资者而言，英特尔封装产能扩产仍会增加设备采购，但这个逻辑太遥远，暂不作为核心跟踪项。

## 5.5 海外公司总览对比

| 公司 | 模式 | 核心封装平台 | 全球封装排名 | 投资意义（A 股关联） |
|------|------|------------|------------|-------------------|
| **台积电** | Foundry | CoWoS/InFO/SoIC | 第 1 | 行业景气度风向标；外溢效应拉动 A 股封装和设备 |
| **日月光** | OSAT | VIPack | 第 2 | 外溢订单的主要受益者；验证行业景气度 |
| **三星** | IDM | I-Cube/X-Cube | — | HBM 拉动设备/材料需求 |
| **英特尔** | IDM | EMIB/Foveros | — | 远期关注（IDM 2.0 成败决定封装价值） |
