# 第四章：A股核心公司分析

---

## 4.1 长电科技（600584.SH）

### 基本面

| 维度 | 信息 |
|------|------|
| **定位** | 中国大陆封装行业绝对龙头，全球第 3（市占率 12%），技术覆盖最全面的综合性封测平台 |
| **2025 年营收** | 约 388.7 亿元（+8.09%），创历史新高 |
| **2025 年净利润** | 15.65 亿元（-2.75%），增收不增利 |
| **2024 年营收** | 359.6 亿元（+21.24%） |
| **2025Q1 营收** | 93.35 亿元（+36.44%），增速显著加速 |
| **毛利率** | ~18-20%（封装行业整体偏低，但先进封装占比提升有望拉动毛利率上行） |
| **全球份额** | 12%（2024 年） |

### 业务结构

长电的营收来自四大封装平台：
- **传统封装**：占比约 50%+，基本盘稳定但增速低
- **先进封装**（2.5D/3D/Fan-out/SiP）：占比约 30%+，增速最快，是公司未来增长的核心
- **测试服务**：占比约 15%
- 公司 2024 年 2.5D/3D 封装产线产能利用率持续高位，先进封装收入增速显著高于整体营收增速

### 客户结构

| 客户 | 关系深度 | 对公司的影响 |
|------|---------|------------|
| **高通** | 长期深度合作 | 稳定贡献 FC-BGA/Fan-out 订单 |
| **博通** | 主要封装伙伴 | 网络芯片封装需求稳定 |
| **AMD** | 封装供应商之一（非唯一） | 部分 CPU/GPU 封装订单 |
| **华为海思** | 国内核心封装伙伴 | Chiplet 设计 + 封装协同 |
| **英伟达** | 部分产品封装 | 体量不大但战略意义强 |

> 相比通富微电绑定单一客户（AMD），长电的客户结构更分散、更全球化。好处是抗风险能力强，坏处是缺少「一家客户暴发就全部吃下」的爆发力。

### 全球化布局

| 基地 | 产能定位 |
|------|---------|
| **中国（江阴/宿迁/滁州）** | 总部，覆盖全品类，先进封装主力基地 |
| **韩国（长电韩国）** | 服务韩国存储芯片客户（三星/SK 海力士） |
| **新加坡（STATS ChipPAC）** | 服务国际客户，规避地缘政治风险 |

> 新加坡基地是长电在中美科技博弈中的关键筹码——可以承接一部分不想在中国大陆封装的国际订单。

### 投资逻辑

| 逻辑 | 详细说明 |
|------|---------|
| **规模优势** | 营收约等于通富+华天之和，规模效应带来成本优势和议价能力 |
| **Chiplet 生态最大受益方** | 长电是 UCIe 标准的核心参与者，Chiplet 越开放、越标准化，长电的平台价值越大 |
| **封装外溢承接方** | 台积电 CoWoS 产能溢出，日月光独占最大份额，但长电也在逐步接到外溢订单 |
| **国产替代主阵地** | 华为昇腾/海思的 Chiplet 方案需要本土先进封装支持，长电是首选 |

### 风险

- 封装制造业利润率天然偏低（~15-20%），长期利润弹性不如上游设备/材料
- 2.5D/3D 高端封装的技术难度大，与台积电/日月光的技术差距仍然存在
- 全球化布局意味着海外资产的地缘政治风险

## 4.2 通富微电（002156.SZ）

### 基本面

| 维度 | 信息 |
|------|------|
| **定位** | 国内第 2，全球前 5，AMD 最核心的封测合作伙伴 |
| **2025 年营收** | 约 279.2 亿元（+16.92%）——增速显著回升 |
| **2025 年净利润** | 12.19 亿元（+79.86%）——利润爆发式增长 |
| **2024 年营收** | 238.8 亿元（+7.24%） |
| **核心客户** | AMD（占比超过 50%+），正在拓展英伟达等其他客户 |

### 业务特色

通富微电与同行最大的不同是**深度绑定 AMD**。这种关系是双刃剑：
- **好处**：AMD 的 CPU/GPU 卖得越好，通富微电的订单越稳定。AI 驱动的算力需求拉动了整个芯片产业链，AMD 作为英伟达之后第二梯队的受益者，封测需求自然增长。
- **坏处**：营收和利润高度依赖 AMD 的经营节奏。如果 AMD 某季度芯片出货不如预期，或者 AMD 部分转单其他封装厂，通富微电受影响直接。

### 先进封装定位

通富微电在 Chiplet 封装上有先发优势——AMD 从 Zen 2（2019 年）开始大搞 Chiplet，通富微电全程参与，积累了丰富的多芯粒集成封装经验。但这一优势建立在 AMD 持续领跑 Chiplet 路线的前提下。

### 投资逻辑

| 逻辑 | 详细说明 |
|------|---------|
| **AMD 影子股票** | 买 AMD 在中国买不到？买通富微电相当于买 AMD 在中国封测环节的映射 |
| **确定性强** | 大客户订单稳定，经营波动的方向可预测（跟着 AMD 财报走） |
| **Chiplet 经验积累** | 深度参与 AMD 的多代 Chiplet 产品封装，经验壁垒存在 |

### 风险

| 风险 | 严重程度 |
|------|---------|
| **单一大客户依赖** | 高（AMD 营收占比超过 50%+） |
| **增速偏慢** | 低（2025 年已从 7.24% 加速至 16.92%，势头好转） |
| **客户拓展进度不确定** | 中（英伟达订单尚在拓展中，体量未明朗） |

## 4.3 华天科技（002185.SZ）

### 基本面

| 维度 | 信息 |
|------|------|
| **定位** | 国内第 3，稳健追赶者 |
| **2025 年营收** | 约 172.1 亿元（+19.0%） |
| **2025 年净利润** | 7.11 亿元（+15.3%），扣非净利润 2.0 亿（+499.8%）——主业盈利质量大幅改善 |
| **2024 年营收** | 144.6 亿元（+28%） |
| **2026Q1 净利润** | 8679 万元（+568.39%）——利润拐点确认 |
| **毛利率** | ~20-22%（成本控制优于同行） |

### 业务特色

| 特色 | 说明 |
|------|------|
| **成本优势** | 产能集中在中国内陆（天水/西安），人工成本和运营成本低于沿海同行 |
| **存储封装赛道** | 在存储芯片封装（DRAM/NAND）上有特色工艺积累，未来 HBM 封装需求增长时有望受益 |
| **客户结构分散** | 以国内芯片设计公司为主，不依赖单一客户——抗风险能力比通富强，但缺少「暴发弹性」 |
| **规模化生产** | 专注中高端封装的大规模量产，良率和效率控制是核心竞争力 |

### 先进封装布局

华天在先进封装上的技术水平不及长电和通富（尤其在 2.5D/3D 封装上），但在 Fan-out 和 WLCSP 等中高端封装上产能充足、良率高。公司过去几年的核心策略是：**先在成熟先进封装上做到极致的成本优势，然后逐步向更高端技术延伸**。

### 投资逻辑

| 逻辑 | 详细说明 |
|------|---------|
| **国产替代弹性最大** | 营收基数最小（145 亿 vs 长电 360 亿），国产化率每提升 1%，对华天的营收弹性最大 |
| **成本护城河** | 内陆建厂 + 规模化生产 → 毛利率优于同行，在价格竞争中有安全垫 |
| **存储封装顺风** | HBM 需求暴增带动存储封装需求，华天在存储封装上有工艺积累 |

### 风险

- 2.5D/3D 高端封装技术差距最大，短期内难以承接 AI 芯片的高端封装订单
- 增速虽然快，但高增速是否能持续取决于国产化替代的实际节奏

## 4.4 A 股封装三巨头横向对比

| 维度 | 长电科技 | 通富微电 | 华天科技 |

| **净利润** | 15.65 亿 | 12.19 亿 | 7.11 亿 |
| **净利增速** | -2.75% | +79.86% | +15.3% |
| **全球排名** | 第 3 | 前 5 | 前 10 |
| **客户结构** | 全球化、分散 | AMD 深度绑定 | 国内为主、分散 |
| **技术全面性** | ★★★★★ | ★★★★ | ★★★ |
| **Chiplet 受益程度** | ★★★★★ | ★★★★ | ★★★ |
| **增长弹性** | ★★★ | ★★★★ | ★★★★★ |
| **经营稳定性** | ★★★★★ | ★★ | ★★★ |
| **适合什么风格** | 求稳的底仓 | 看好 AMD 的映射 | 博国产替代弹性 |

## 4.5 上游材料与设备代表

### 雅克科技（002409.SZ）

| 维度 | 信息 |
|------|------|
| **定位** | 半导体材料平台型公司，国内先进封装前驱体/电子特气龙头 |
| **核心产品** | 前驱体材料（用于薄膜沉积）、电子特气、光刻胶 |
| **客户** | 覆盖长电科技、通富微电、华天科技等国内主要封测厂，部分导入台系客户 |
| **看点** | 前驱体国产替代空间大（日/美系占主导）；材料品类持续扩展 |

### 上海新阳（300236.SZ）

| 维度 | 信息 |
|------|------|
| **定位** | 半导体电镀液与清洗液龙头 |
| **核心产品** | 电镀液（TSV/RDL 工艺关键耗材）、清洗液、光刻胶 |
| **看点** | 电镀液是先进封装不可或缺的材料，技术壁垒高、替换成本高；光刻胶布局加深 |
| **国产化率** | 电镀液国产化率不到 20%，上海新阳是国内最大玩家 |

### 北方华创（002371.SZ）

| 维度 | 信息 |
|------|------|
| **定位** | 国产半导体设备绝对龙头，覆盖刻蚀、薄膜沉积、热处理等 |
| **2025 年营收** | 393.53 亿元（+30.85%） |
| **2025 年净利润** | 55.22 亿元（-1.77%），首次出现增收不增利 |
| **在先进封装中的位置** | 刻蚀机（TSV 通孔刻蚀）、PVD 设备（金属薄膜沉积）、炉管设备——均为封装产线的核心设备 |
| **看点** | 不挑「谁在封装」——无论台积电/日月光/长电谁扩产，设备都得买；国产化率提升趋势明确 |
| **注意事项** | 封装设备收入占比不高（晶圆制造设备是绝对主力），先进封装对其业绩的边际拉动有限但确定性高 |
