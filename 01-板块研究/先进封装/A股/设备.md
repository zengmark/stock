# A股先进封装 — 上游设备

设备是先进封装产能的「先行指标」——封装厂扩产前必先采购设备，因此设备订单是先于封装业绩的前瞻信号。先进封装对设备的需求同时在「量」和「质」两个维度上提升：不仅需要更多数量的设备，还需要精度更高、功能更复杂的设备。

---

## 一、刻蚀与薄膜沉积设备

### 北方华创（002371.SZ）— 平台型设备龙头

| 维度 | 信息 |
|------|------|
| **定位** | 国产半导体设备绝对龙头，覆盖刻蚀、薄膜沉积、热处理、清洗等 |
| **2025 年营收** | 393.53 亿元（<span style="color:red">+30.85%</span>） |
| **2025 年净利润** | 55.22 亿元（<span style="color:green">-1.77%</span>），首次增收不增利 |
| **主营构成** | 电子工艺装备 367.31 亿（93.34%）、电子元器件 25.79 亿（6.55%） |
| **先进封装中的设备** | 刻蚀机（TSV 通孔深硅刻蚀）、PVD 设备（阻挡层/种子层金属沉积）、炉管设备（氧化/退火） |
| **看点** | 不挑「谁在封装」——台积电/日月光/长电谁扩产，设备都得买；平台型产品线覆盖多个工艺环节 |
| **注意事项** | 封装设备收入占比不高（晶圆制造设备是绝对主力），先进封装对业绩的边际拉动有限但确定性强 |

### 中微公司（688012.SH）— 刻蚀专精龙头

| 维度 | 信息 |
|------|------|
| **定位** | 国产等离子体刻蚀设备龙头，技术最接近国际水平的国产设备商 |
| **2025 年营收** | 123.85 亿元（<span style="color:red">+36.62%</span>），归母净利润 21.11 亿元（<span style="color:red">+30.69%</span>） |
| **主营构成** | 半导体设备 123.85 亿（100%，含刻蚀设备与 MOCVD 两大产品线） |
| **核心产品** | CCP 刻蚀机（介质刻蚀）、ICP 刻蚀机（硅刻蚀）、MOCVD（LED/化合物半导体外延设备，公司第二大产品线） |
| **先进封装关联** | TSV 工艺需要深硅刻蚀（Bosch 工艺）——中微的 ICP 刻蚀机在 TSV 刻蚀上有针对性的产品布局 |
| **壁垒** | 深硅刻蚀要求高深宽比（>10:1），对刻蚀速率/均匀性/侧壁垂直度的要求远超常规刻蚀 |
| **进度** | TSV 刻蚀设备已在部分封测厂通过验证，处于放量初期阶段 |

### 拓荆科技（688072.SH）— PECVD 龙头

| 维度 | 信息 |
|------|------|
| **定位** | 国产 PECVD（等离子体增强化学气相沉积）设备龙头 |
| **核心产品** | PECVD、SACVD、HDPCVD |
| **先进封装关联** | RDL 工艺中的介电层沉积、TSV 侧壁钝化层沉积均需要 CVD 设备；多层 RDL 每增加一层就多用一轮 CVD |
| **看点** | 先进封装 RDL 层数增加→CVD 设备用量增加；公司 CVD 产品线从晶圆制造向先进封装延伸 |

## 二、电镀与清洗设备

### 盛美上海（688082.SH）— 清洗+电镀双龙头

| 维度 | 信息 |
|------|------|
| **定位** | 国产半导体清洗设备龙头，电镀设备快速追赶 |
| **核心产品** | 单片清洗设备、电镀设备（铜互连电镀） |
| **先进封装关键性** | TSV 通孔形成后需要铜电镀填充（高深宽比电镀是核心技术难点）；每道工艺前后的清洗步骤都需要清洗设备 |
| **电镀壁垒** | TSV 电镀要求从通孔底部向上均匀填充（Bottom-up Fill），防止空洞形成——添加剂的配方和电镀参数控制是核心壁垒 |
| **看点** | 电镀设备+清洗设备+前道涂胶显影设备，产品组合覆盖先进封装多个关键工艺环节 |

## 三、CMP 与平坦化设备

### 华海清科（688120.SH）— CMP 设备国产独苗

| 维度 | 信息 |
|------|------|
| **定位** | 国产 CMP 设备唯一规模化供应商 |
| **核心产品** | CMP 设备（铜/钨/氧化物抛光） |
| **先进封装关联** | TSV 铜电镀后 CMP 平坦化、RDL 层间平坦化——每增加一层 RDL，多用一轮 CMP。先进封装的 CMP 需求甚至超过了部分前道工艺 |
| **格局** | CMP 设备全球市场应用材料（AMAT）和荏原制作所（Ebara）垄断 90%+，华海清科是唯一打破垄断的国产玩家 |
| **看点** | 技术壁垒极高（CMP 是物理+化学复合过程，控制参数几十个），国产替代确定性高 |

## 四、涂胶显影与其他设备

### 芯源微（688037.SH）— 涂胶显影龙头

| 维度 | 信息 |
|------|------|
| **定位** | 国产涂胶显影设备龙头，向先进封装拓展 |
| **核心产品** | 涂胶显影设备（Track）、去胶设备 |
| **先进封装关联** | RDL 图案化需要光刻胶涂布→曝光→显影，涂胶显影设备是光刻工艺的配套刚需；Fan-out 封装同样需要涂胶显影步骤 |
| **壁垒** | 涂胶均匀性（±1%）直接影响后续曝光质量，设备精度要求极高 |
| **看点** | 先进封装工序中涂胶显影步骤比前道晶圆制造更多，设备用量大 |

### 长川科技（300604.SZ）— 测试设备

| 维度 | 信息 |
|------|------|
| **定位** | 国产半导体测试设备龙头 |
| **核心产品** | 测试机（ATE）、分选机（Handler）、探针台（Prober） |
| **先进封装关联** | 先进封装复杂度高→需要更多测试环节→测试设备用量增加。Chiplet 封装后需要做 KGD（Known Good Die）测试，一颗多芯粒封装在最终测试前需要先测试每个芯粒，测试量大增 |
| **看点** | 先进封装的「系统级测试」复杂度远超传统封装，长川在模拟/混合信号测试机上有优势 |

### 华峰测控（688200.SH）— 模拟测试机

| 维度 | 信息 |
|------|------|
| **定位** | 模拟/混合信号测试机龙头 |
| **核心产品** | 模拟测试机（STS 系列） |
| **与长川的关系** | 长川覆盖数字+模拟+分选+探针台全线，华峰专精模拟测试机；两者形成互补而非直接竞争 |

## 五、设备环节投资地图

| 工艺环节 | 关键设备 | 代表公司 | 国产化率 | 先进封装新增量 |
|---------|---------|---------|---------|-------------|
| **TSV 刻蚀** | 深硅刻蚀机 | 中微公司、北方华创 | ~20% | 高（每增加 TSV 刻蚀量增加） |
| **TSV 填充** | 铜电镀设备 | 盛美上海 | ~15% | 高（TSV 数量驱动） |
| **RDL 薄膜沉积** | PVD/CVD 设备 | 北方华创、拓荆科技 | ~15% | 高（RDL 层数驱动） |
| **CMP 平坦化** | CMP 设备 | 华海清科 | ~5% | 高（RDL 层数驱动） |
| **光刻配套** | 涂胶显影 | 芯源微 | ~15% | 中（RDL 层数驱动） |
| **晶圆减薄** | 减薄机 | 华海清科正在布局 | ~5% | 中（3D 封装必需） |
| **晶圆键合** | 键合设备 | — | （国产近乎空白） | 高（3D 堆叠核心设备） |
| **测试** | ATE/分选/探针 | 长川、华峰 | ~20% | 高（复杂度驱动） |

> **核心观察**：先进封装设备国产化率整体偏低（多数在 15-20%），最大的国产替代空间在刻蚀、沉积、电镀、CMP 四大环节。晶圆键合设备目前国产近乎空白，是「卡脖子」程度最严重的环节，但也是潜在爆发空间最大的方向。
