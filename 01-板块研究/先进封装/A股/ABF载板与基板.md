# A股先进封装 — ABF 载板与封装基板

封装基板是先进封装的「地基」——芯片和硅中介层放在基板上，基板再通过 BGA 焊球连到 PCB。先进封装对基板的要求远高于传统封装：更多层数、更细线宽线距、更大尺寸。ABF（Ajinomoto Build-up Film）载板是 FC-BGA 和 2.5D 封装的核心基板材料。

---

## 一、ABF 载板：为什么重要？

### 技术定位

ABF 载板是用味之素（Ajinomoto）公司开发的增层膜材料制造的 IC 载板，特点是：

- **高密度布线**：线宽/线距可达 10μm/10μm 级别（传统 PCB 通常是 50μm+）
- **多层结构**：高端 ABF 载板可达 14-20 层
- **大尺寸**：2.5D 封装需要承载多颗芯片 + 硅中介层，基板面积是传统单芯片封装的 3-5 倍

### 需求驱动

每一颗 FC-BGA 或 2.5D 封装芯片都需要一块 ABF 载板。以 AI 芯片为例：

- 英伟达 H100：1 颗 GPU + 6 颗 HBM → 1 块大尺寸 ABF 载板
- AMD MI300X：1 颗 GPU + 8 颗 HBM → 1 块超大尺寸 ABF 载板

> **简洁逻辑**：AI 芯片出货量 × 每颗芯片的 ABF 载板用量 = ABF 载板总需求。AI 芯片在暴增，大尺寸 ABF 载板在暴增。

## 二、核心公司

### 深南电路（002916.SZ）— ABF 载板旗舰

| 维度 | 信息 |
|------|------|
| **定位** | 国内高端 PCB 和封装基板龙头，ABF 载板国产替代核心标的 |
| **2025 年营收** | ~167 亿元 |
| **封装基板业务** | 无锡/南通/广州三大基地，FC-BGA 封装基板已进入量产 |
| **客户** | 华为海思、中兴微电子、国内主要封测厂 |
| **核心看点** | 国内 ABF 载板自给率不到 5%，深南电路是唯一具备大规模量产能力的国产厂商 |
| **扩产进度** | 广州基地 FC-BGA 项目按计划推进，预计 2025-2026 年逐步贡献增量产能 |
| **技术差距** | 与日系（揖斐电 Ibiden、新光电工 Shinko）在高层数/细线距方面仍有差距，但在 8-12 层产品上已接近 |
| **风险** | 高端 ABF 载板产能爬坡周期长（2-3 年），短期业绩贡献有限；日系竞争对手也在大幅扩产 |

### 兴森科技（002436.SZ）— ABF 载板第二极

| 维度 | 信息 |
|------|------|
| **定位** | PCB 样板龙头，积极转型封装基板 |
| **封装基板进展** | 广州基地 FC-BGA 封装基板项目推进中，总投资约 60 亿元 |
| **当前阶段** | 从样板/快板向批量基板转型中，先进封装基板大规模业绩贡献尚需时间 |
| **与深南的差异** | 兴森在 PCB 样板领域积累的快速响应能力是独特优势，但基板批量生产能力仍在建设中 |
| **看点** | 国产 ABF 载板玩家稀少，兴森作为第二家有量产布局的厂商，稀缺性明显 |

### 景旺电子（603228.SH）

| 维度 | 信息 |
|------|------|
| **定位** | 国内刚性/柔性 PCB 龙头，切入封装基板领域 |
| **进展** | 珠海基地布局类载板（SLP）和封装基板产能 |
| **阶段** | 封装基板收入占比很小，属于远期布局 |

### 崇达技术（002815.SZ）

| 维度 | 信息 |
|------|------|
| **定位** | 中小批量 PCB 厂商，布局 IC 载板 |
| **进展** | 珠海崇达 IC 载板项目在建 |
| **阶段** | 早期建设阶段，短期无显著业绩贡献 |

## 三、ABF 载板投资地图

| 公司 | 当前阶段 | 确定性 | 弹性 | 适合逻辑 |
|------|---------|--------|------|---------|
| **深南电路** | 量产 | ★★★★ | ★★★ | 国产替代确定性最强的底仓 |
| **兴森科技** | 产能建设 | ★★ | ★★★★★ | 博放量弹性，但需要耐心 |
| **景旺电子** | 早期布局 | ★ | ★★★ | 远期布局，当前影响有限 |
| **崇达技术** | 初始建设 | ★ | ★★★ | 同上 |

> **关键跟踪指标**：深南和兴森的 FC-BGA 基地产能利用率、ABF 载板产品导入的客户数量、ABF 膜供应商味之素（Ajinomoto）的产能扩张进度（ABF 膜的供给也会限制载板产能）。
