# A股先进封装 — Chiplet 设计与 EDA 工具

Chiplet 是先进封装的下半场。如果说 2.5D/3D 封装是「怎么把芯片拼在一起」，那 Chiplet 就是「拼成什么样的系统」。Chiplet 生态的成熟会催生对设计工具（EDA）、IP 核和系统集成服务的需求，A 股有一批公司直接受益。

---

## 一、Chiplet 设计服务与 IP

### 芯原股份（688521.SH）— Chiplet IP 平台

| 维度 | 信息 |
|------|------|
| **定位** | 中国最大的芯片设计服务（Design Service）和 IP 授权公司 |
| **Chiplet 定位** | 公司明确将 Chiplet 作为核心战略方向，提供 Chiplet 设计服务 + Chiplet IP 平台 |
| **核心能力** | 拥有丰富的 IP 库（GPU/VPU/NPU/DSP/存储器接口等），可以将不同功能的 IP 核封装为 Chiplet，帮助客户快速搭建芯片系统 |
| **UCIe 参与** | 芯原是 UCIe 标准的正式成员（Contributor Member），参与制定 Chiplet 互联标准 |
| **看点** | 如果 Chiplet 生态走向开放化/标准化，芯原作为中立 IP 平台将是大赢家——客户可以像搭积木一样从芯原挑选 Chiplet，拼成定制芯片 |
| **风险** | Chiplet 生态的开放化进程不确定，目前多数 Chiplet 方案仍是大厂（AMD/苹果）的闭门独享方案，第三方 IP 平台的实际商业落地面尚未大规模展开 |

### 国芯科技（688262.SH）

| 维度 | 信息 |
|------|------|
| **定位** | 国产嵌入式 CPU IP 核供应商 |
| **Chiplet 关联** | 公司的 CPU IP 可以作为 Chiplet 系统中的主控核心，通过 Chiplet 方式集成到更大的芯片系统中 |
| **阶段** | 在 Chiplet 领域有技术布局和技术储备，但尚未形成 Chiplet 业务的规模收入 |

## 二、EDA 工具

### 华大九天（301269.SZ）— 国产 EDA 旗舰

| 维度 | 信息 |
|------|------|
| **定位** | 国产 EDA（电子设计自动化）工具绝对龙头 |
| **核心产品** | 模拟电路全流程 EDA、数字电路部分环节 EDA、平板显示全流程 EDA |
| **先进封装关联** | 先进封装（尤其是 2.5D/3D 多芯片集成）需要专门的 EDA 工具来设计芯片间互联、信号完整性分析、热仿真、电源完整性分析——传统 PCB 工具和芯片设计工具都覆盖不了这个「中间地带」 |
| **封装 EDA 布局** | 公司正在开发先进封装专用 EDA 工具，覆盖 2.5D/3D 封装的布局布线和电热仿真 |
| **看点** | 先进封装设计复杂度呈指数级增长，对 EDA 的需求不是「多一两个功能」而是「需要一套全新的工具链」。华大九天作为国产 EDA 唯一平台型公司，封装 EDA 是重要增长极 |
| **竞争格局** | 全球先进封装 EDA 市场被 Cadence/Synopsys 垄断，国产化率近乎零——替代空间极大，但追赶难度也极大 |

### 概伦电子（688206.SH）

| 维度 | 信息 |
|------|------|
| **定位** | 国产 EDA 工具，专注器件建模和电路仿真 |
| **核心产品** | SPICE 模型、器件建模工具、电路仿真工具 |
| **先进封装关联** | 3D 封装中 TSV 的结构参数、电气特性需要通过仿真来分析和优化——概伦的 SPICE 建模和仿真能力直接对应这个需求 |
| **看点** | 先进封装需要更精确的互连模型（TSV 的电感/电容/电阻参数提取），概伦在模型精度方面有独特优势 |

## 三、Chiplet 生态投资地图

| 公司 | 定位 | 与 Chiplet 关联 | 受益阶段 |
|------|------|---------------|---------|
| **芯原股份** | IP + 设计服务 | 提供 Chiplet 设计和 IP 平台 | Chiplet 生态规模化放量后 |
| **华大九天** | EDA 平台 | 先进封装 EDA 工具 | 封装设计复杂度提升持续受益 |
| **概伦电子** | 建模/仿真 | TSV 建模与仿真 | 3D 封装普及阶段 |
| **国芯科技** | CPU IP | CPU IP 可集成入 Chiplet | 远期 |

> **投资者关键认知**：Chiplet 对 EDA/IP 公司的拉动是「二阶导数」逻辑——先有芯片设计公司用 Chiplet 做产品，再有封装厂接到 Chiplet 订单，最后才是 EDA/IP 工具的采购增加。短期弹性不如设备和材料，但长期天花板极高（类比：没有 EDA 就没有芯片设计，没有先进的封装 EDA 就没有 Chiplet 设计）。
