# A股先进封装 — 上游材料与耗材

先进封装的价值链中，材料环节是"卖铲子"逻辑最集中的区域——不管谁赢、不管哪家封装厂胜出，材料都得用。而且先进封装对材料的需求在**量和价**两个维度上都在提升：用量更多、单价更贵。

---

## 一、核心材料：前驱体与电子特气

### 雅克科技（002409.SZ）— 前驱体平台龙头

| 维度 | 信息 |
|------|------|
| **定位** | 半导体材料平台型公司，国内先进封装前驱体/电子特气龙头 |
| **核心产品** | 前驱体材料（用于 ALD/CVD 薄膜沉积）、电子特气（高纯氟碳类）、光刻胶 |
| **客户** | 长电科技、通富微电、华天科技等国内主要封测厂，部分导入台系客户 |
| **在先进封装中的位置** | TSV 工艺中需要在通孔内沉积阻挡层/种子层 → 前驱体是 ALD 工艺核心耗材；RDL 工艺中薄膜沉积同样大量消耗前驱体 |
| **国产化率** | 前驱体市场日美系厂商（默克/液化空气/ADEKA）占主导，国产化率不到 15%，雅克是国内唯一规模化供应商 |
| **壁垒** | 前驱体是定制化化学合成品，每种材料对应特定工艺配方，认证周期长（1-2 年），客户粘性极高 |

> **关键逻辑**：前驱体的生意模式类似"剃须刀片"——设备（CVD/ALD）卖一次，前驱体持续消耗。先进封装扩产阶段，前驱体用量增长快于设备台数增长（单片晶圆消耗量不变，但总产能增加）。

### 南大光电（300346.SZ）

| 维度 | 信息 |
|------|------|
| **定位** | MO 源（金属有机化合物）龙头，电子特气新秀 |
| **核心产品** | MO 源（LED/化合物半导体外延关键材料）、磷烷/砷烷（掺杂气体）、ArF 光刻胶（193nm） |
| **先进封装关联** | 电子特气在 TSV 刻蚀/清洁环节中作为工艺气体使用；光刻胶可用于先进封装的 RDL 图案化 |

## 二、核心材料：电镀液与 CMP 耗材

### 上海新阳（300236.SZ）— 电镀液龙头

| 维度 | 信息 |
|------|------|
| **定位** | 半导体电镀液与清洗液龙头 |
| **核心产品** | 电镀液（TSV/RDL 工艺关键耗材）、清洗液、光刻胶 |
| **先进封装关键性** | TSV 通孔形成后需要铜电镀填充，RDL 重布线层也需要电镀形成铜互连——电镀液是这两道核心工艺的必需耗材 |
| **国产化率** | 电镀液国产化率不到 20%，上海新阳是国内最大玩家 |
| **壁垒** | 电镀液配方中添加剂（平整剂/抑制剂/加速剂）是关键 Know-how，竞争对手难以逆向模仿；更换电镀液品牌意味着重新调校整条产线——替换成本极高 |
| **客户** | 覆盖国内主要封测厂，部分进入国际供应链 |

### 安集科技（688019.SH）— CMP 抛光液龙头

| 维度 | 信息 |
|------|------|
| **定位** | 国产 CMP（化学机械抛光）抛光液绝对龙头 |
| **核心产品** | 铜/钨/阻挡层抛光液、清洗液 |
| **先进封装关联** | TSV 铜电镀后需要通过 CMP 去除多余的铜并平坦化表面；RDL 层间同样需要 CMP 平坦化——每增加一层 RDL，多用一轮 CMP |
| **壁垒** | 抛光液中磨料粒径/分布直接影响抛光质量和良率，配方案高度保密；与客户工艺深度绑定 |
| **国产化率** | 铜抛光液国产化率 ~30%，钨抛光液 ~10%，安集是铜抛光液唯一国产规模化供应商 |

### 鼎龙股份（300054.SZ）— CMP 抛光垫龙头

| 维度 | 信息 |
|------|------|
| **定位** | CMP 抛光垫国产唯一规模化供应商 |
| **核心产品** | CMP 抛光垫（与安集的抛光液是配套耗材，一对一的消耗关系） |
| **格局** | 全球抛光垫市场陶氏化学独占 ~80%，鼎龙是唯一打破垄断的国产玩家 |
| **投资关联** | 设备扩产→CMP 用量增加→抛光垫+抛光液同步受益。鼎龙+安集可以看作 CMP 赛道的双子星 |

## 三、核心材料：封装基板与硅片

### 深南电路（002916.SZ）— ABF 载板国产替代

| 维度 | 信息 |
|------|------|
| **定位** | 国内高端 PCB 龙头，ABF 载板重点供应商 |
| **核心业务** | ABF 载板（用于 FC-BGA/2.5D 封装基板）、高端多层 PCB、封装基板 |
| **先进封装关键性** | 2.5D/3D 封装的 ABF 基板是芯片与 PCB 之间的"最后一公里"——不载芯片但承载封装体 |
| **客户** | 华为海思、国内封测厂、部分国际客户 |
| **看点** | 国内 ABF 载板自给率不到 5%，深南电路是少数有量产能力的国产玩家；每一颗先进封装芯片都需要一块 ABF 载板——量价齐升逻辑清晰 |
| **风险** | ABF 载板技术门槛高，与日系（揖斐电/新光电工）品质差距仍在；产能扩张周期长（2-3 年） |

### 兴森科技（002436.SZ）

| 维度 | 信息 |
|------|------|
| **定位** | PCB 样板龙头，加速转型封装基板 |
| **核心进展** | 广州基地 FC-BGA 封装基板项目推进中，是国内 ABF 载板第二极 |
| **阶段** | 仍处于从样板厂商向批量基板厂商转型过程中，先进封装基板对业绩的贡献尚未大规模体现 |

### 沪硅产业（688126.SH）

| 维度 | 信息 |
|------|------|
| **定位** | 国产大硅片龙头 |
| **先进封装关联** | 2.5D 封装需要硅中介层——硅中介层本质是一片高平整度的 300mm 硅片，上面做布线。沪硅是台积电 CoWoS 用高平整度硅片的潜在供应商 |
| **逻辑** | 硅中介层需求暴增→硅片需求增加→沪硅受益。但要注意硅中介层对硅片的要求（翘曲度/平整度）远高于普通硅片，技术门槛更高 |

## 五、材料环节投资地图

| 细分赛道 | 国产化率 | 代表公司 | 受益环节 | 确定性 |
|---------|---------|---------|---------|--------|
| **前驱体** | <15% | 雅克科技 | TSV/RDL 薄膜沉积 | ★★★★ |
| **电镀液** | <20% | 上海新阳 | TSV/RDL 铜电镀 | ★★★★ |
| **CMP 抛光液** | ~30% | 安集科技 | TSV/RDL 平坦化 | ★★★★★ |
| **CMP 抛光垫** | <10% | 鼎龙股份 | TSV/RDL 平坦化 | ★★★★ |
| **ABF 载板** | <5% | 深南电路 | 封装基板 | ★★★★★ |
| **硅片** | ~15% | 沪硅产业 | 硅中介层 | ★★★ |
| **封装胶/填充胶** | <10% | 飞凯/德邦 | FC 封装/2.5D 底部填充 | ★★★ |
| **掩膜版** | ~20% | 清溢光电 | RDL 光刻 | ★★★ |

> **投资优先级的核心判断**：先进封装每增加一层 RDL，CMP、电镀、前驱体、光刻胶的消耗就多一轮。TSV 数量越多，电镀液和前驱体消耗越大。因此材料环节是封装"量×复杂度"双轮驱动的赛道。
