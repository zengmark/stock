# 先进封装 A 股核心公司分析

> 本文为《先进封装行业研究》第 4 章的展开，聚焦 A 股先进封装产业链核心标的。

---

## 一、封测三巨头

### 长电科技（600584.SH）

**一句话定位**：中国封装行业绝对龙头，全球第 3（市占率 12%），技术覆盖最全面的综合性封测平台。

| 维度 | 信息 |
|------|------|
| 2024 年营收 | 约 359.6 亿元（+21.24%） |
| 净利润 | 增长显著（2025Q1 净利增超 50%） |
| 核心客户 | 高通、博通、AMD、华为海思、英伟达（部分） |
| 先进封装技术 | 2.5D/3D、Fan-out、SiP 系统级封装、WLCSP |
| 产能布局 | 中国（江阴/宿迁/滁州）+ 韩国 + 新加坡 |
| 全球份额 | 12%（2024 年，较 2023 年的 10.4% 提升） |

**投资逻辑**：
- 规模优势：国内唯一进入全球前三的封测企业，营收体量遥遥领先
- 全球化布局：新加坡/韩国产能使其在中美科技博弈中更有弹性
- Chiplet 生态受益者：作为中立 OSAT，如果 Chiplet 标准 UCIe 生态扩大，长电是最大受益方

**风险提示**：
- 相比通富微电，缺乏 AMD 这样高度绑定的单一核心客户，客户结构分散可能影响订单稳定性

---

### 通富微电（002156.SZ）

**一句话定位**：AMD 最核心封测伙伴，AI 芯片产业链的「影子受益者」。

| 维度 | 信息 |
|------|------|
| 2024 年营收 | 约 238.8 亿元（+7.24%） |
| 核心客户 | AMD（占比最高，深度绑定）、英伟达（部分，正在拓展） |
| 先进封装技术 | 2.5D 封装、FC-BGA、Chiplet 封装 |
| 产能布局 | 国内为主，海外配合 AMD 节奏布局 |
| 特色优势 | 与 AMD 联合研发能力强，在 Chiplet 封装上领先国内同业 |

**投资逻辑**：
- AMD 对标英伟达：AMD 的 CPU/GPU 需要先进封装，通富微电是其核心封测伙伴
- AI 映射：英伟达 GPU 卖得越好 → AI 算力需求越大 → AMD 对标产品需求也越大 → 通富微电受益
- Chiplet 先发优势：AMD 是 Chiplet 路线的先驱（从 Zen 2 开始），通富微电与之深度协同，积累了丰富的多芯粒封装经验

**风险提示**：
- **大客户依赖风险**：营收高度依赖 AMD，若 AMD 市占率变化或转单其他封装厂，影响直接
- 增速偏慢（7.24% 低于长电的 21% 和华天的 28%），需要关注新客户拓展进度

---

### 华天科技（002185.SZ）

**一句话定位**：国内封测第 3，增速最快的追赶者，存储封装有特色。

| 维度 | 信息 |
|------|------|
| 2024 年营收 | 约 144.6 亿元（+28%） |
| 核心客户 | 以本土芯片设计公司为主，客户结构分散 |
| 先进封装技术 | Fan-out、WLCSP、存储芯片堆叠封装 |
| 产能布局 | 西安/天水/南京三地 |
| 特色优势 | 成本控制能力强，规模化生产效率高 |

**投资逻辑**：
- 弹性最大：营收基数在三巨头中最小，增速最快（+28%），国产替代放量时弹性最高
- 存储封装赛道：在存储芯片封测上有差异化优势，HBM 趋势下有望受益
- 国产化受益：客户以国内设计公司为主，契合半导体自主化政策方向

**风险提示**：
- 技术水平与长电/通富有差距，在 2.5D/3D 等最高端封装上仍需追赶
- 产能集中在国内，全球化布局不足

## 二、上游材料与设备

### 雅克科技（002409.SZ）

**定位**：半导体材料平台型公司，先进封装材料核心供应商。

主营先进封装用前驱体材料、光刻胶、电子特气等。

**看点**：受益于国内先进封装产能扩张，材料需求同步增长。客户覆盖长电科技、通富微电、华天科技等国内主要封测厂。

### 上海新阳（300236.SZ）

**定位**：半导体材料细分龙头，电镀液、清洗液等封装关键化学品供应商。

**看点**：先进封装工艺中电镀液是不可或缺的材料，技术壁垒较高，国产替代逻辑清晰。

### 北方华创（002371.SZ）

**定位**：国产半导体设备龙头，覆盖刻蚀、薄膜沉积、清洗等封装关键设备。

**看点**：先进封装扩产最确定的上游受益者。不管谁做封装，都需要设备——北方华创是国内封装厂设备采购的核心选项。但注意其对封装设备收入的依赖度不及晶圆制造设备。

## 三、投资逻辑总结

| 逻辑链 | A 股标的 |
|--------|---------|
| AI 算力爆发 → 先进封装产能紧缺 → 封测龙头受益 | **长电科技**（最大、最全）|
| AMD 对标英伟达 → 芯片量增 → 封测订单增 | **通富微电**（深度绑定 AMD）|
| 国产替代加速 → 国内芯片设计公司增量 → 中立封测受益 | **华天科技**（国内客户为主）|
| 先进封装扩产 → 材料/设备需求同步增长 | **雅克科技、上海新阳、北方华创** |
