# A股先进封装 — 封装厂

先进封装产业链中，封装厂是最直接面向下游客户的环节。A股封装三巨头（长电/通富/华天）分别代表了三种投资逻辑：平台型龙头、大客户绑定、弹性追赶。

---

## 一、长电科技（600584.SH）— 平台型龙头

### 基本面

| 维度 | 信息 |
|------|------|
| **定位** | 中国大陆封装行业绝对龙头，全球第 3（市占率 ~12%），技术覆盖最全面的综合性封测平台 |
| **2025 年营收** | 388.71 亿元（<span style="color:red">+8.09%</span>），创历史新高 |
| **2025 年净利润** | 15.65 亿元（<span style="color:green">-2.80%</span>），增收不增利 |
| **2024 年营收** | 359.6 亿元（<span style="color:red">+21.24%</span>） |
| **2025Q1 营收** | 93.35 亿元（<span style="color:red">+36.44%</span>），增速显著加速 |
| **毛利率** | ~18-20%（封装行业整体偏低，先进封装占比提升有望拉动毛利率上行） |

### 业务结构

长电的营收来自四大封装平台：

- **传统封装**：占比约 50%+，基本盘稳定但增速低
- **先进封装**（2.5D/3D/Fan-out/SiP）：占比约 30%+，增速最快，是公司未来增长核心
- **测试服务**：占比约 15%
- 2024 年 2.5D/3D 封装产线产能利用率持续高位，先进封装收入增速显著高于整体营收增速

### 客户结构

| 客户 | 关系深度 | 对公司的影响 |
|------|---------|------------|
| **高通** | 长期深度合作 | 稳定贡献 FC-BGA/Fan-out 订单 |
| **博通** | 主要封装伙伴 | 网络芯片封装需求稳定 |
| **AMD** | 封装供应商之一 | 部分 CPU/GPU 封装订单 |
| **华为海思** | 国内核心封装伙伴 | Chiplet 设计 + 封装协同 |
| **英伟达** | 部分产品封装 | 体量不大但战略意义强 |

> 相比通富微电绑定单一客户（AMD），长电客户结构更分散、更全球化。抗风险能力强，但缺少「一家客户暴发就全部吃下」的爆发力。

### 全球化布局

| 基地 | 产能定位 |
|------|---------|
| **中国（江阴/宿迁/滁州）** | 总部，全品类覆盖，先进封装主力 |
| **韩国（长电韩国）** | 服务三星/SK 海力士存储客户 |
| **新加坡（STATS ChipPAC）** | 服务国际客户，规避地缘政治风险 |

> 新加坡基地是长电在中美科技博弈中的关键筹码——可承接不想在中国大陆封装的国际订单。

### 投资逻辑

| 逻辑 | 详细说明 |
|------|---------|
| **规模优势** | 营收约等于通富+华天之和，规模效应带来成本优势和议价能力 |
| **Chiplet 生态最大受益方** | UCIe 标准的核心参与者，Chiplet 越开放，长电平台价值越大 |
| **封装外溢承接方** | 台积电 CoWoS 产能溢出，长电逐步接到外溢订单 |
| **国产替代主阵地** | 华为昇腾/海思 Chiplet 方案需要本土先进封装支持，长电是首选 |

### 风险

- 封装制造业利润率天然偏低（~15-20%），长期利润弹性不如上游设备/材料
- 2.5D/3D 高端封装与台积电/日月光的差距依然存在
- 全球化布局意味着海外资产的地缘政治风险

## 二、通富微电（002156.SZ）— AMD 影子股

### 基本面

| 维度 | 信息 |
|------|------|
| **定位** | 国内第 2，全球前 5，AMD 最核心封测合作伙伴 |
| **2025 年营收** | 279.21 亿元（<span style="color:red">+16.92%</span>），增速显著回升 |
| **2025 年净利润** | 12.19 亿元（<span style="color:red">+79.8%</span>），利润爆发式增长 |
| **主营构成** | 集成电路封装测试 272.48 亿（97.59%）、模具及材料销售等 6.74 亿（2.41%） |
| **2024 年营收** | 238.8 亿元（<span style="color:red">+7.24%</span>） |
| **核心客户** | AMD（占比超 50%+），正拓展英伟达等其他客户 |

### 业务特色

通富微电与同行最大的不同是**深度绑定 AMD**：

- **好处**：AMD CPU/GPU 卖得越好，通富订单越稳。AI 算力需求拉动芯片产业链，AMD 作为英伟达之后第二梯队受益者，封测需求自然增长
- **坏处**：营收和利润高度依赖 AMD 经营节奏。如果 AMD 部分转单其他封装厂，通富受影响直接

### 先进封装定位

通富在 Chiplet 封装上有先发优势——AMD 从 Zen 2（2019 年）开始大搞 Chiplet，通富全程参与，积累了丰富的多芯粒集成封装经验。但这一优势建立在 AMD 持续领跑 Chiplet 路线的前提下。

### 投资逻辑与风险

| 逻辑 | 风险 |
|------|------|
| **AMD 影子股**：买 AMD 在中国买不到→买通富 | **单一大客户依赖**（AMD > 50%+）：严重 |
| **确定性强**：大客户订单稳定，可跟踪 | **客户拓展进度不确定**：英伟达订单尚在初期 |
| **Chiplet 经验深厚**：深度参与 AMD 多代 Chiplet | **增速波动**：2025 年已好转（+16.92%），但持续性需观察 |

## 三、华天科技（002185.SZ）— 成本护城河 + 存储赛道

### 基本面

| 维度 | 信息 |
|------|------|
| **定位** | 国内第 3，内陆成本优势型封测厂 |
| **2025 年营收** | 172.14 亿元（<span style="color:red">+19.03%</span>） |
| **2025 年净利润** | 7.11 亿元（<span style="color:red">+15.30%</span>），扣非 2.0 亿（<span style="color:red">+499.8%</span>） |
| **2024 年营收** | 144.6 亿元（<span style="color:red">+28%</span>） |
| **2026Q1 净利润** | 8679 万元（<span style="color:red">+568.39%</span>），利润拐点确认（neodata 未收录 2026Q1，数据待核实） |
| **毛利率** | ~20-22%（成本控制优于同行） |

### 业务特色

| 特色 | 说明 |
|------|------|
| **成本优势** | 产能集中在中国内陆（天水/西安），人工和运营成本低于沿海同行 |
| **存储封装赛道** | 在 DRAM/NAND 封装上有特色工艺积累，HBM 封装需求增长时有望受益 |
| **客户结构分散** | 以国内芯片设计公司为主，不依赖单一客户——抗风险强但缺「暴发弹性」 |
| **规模化生产** | 专注中高端封装大规模量产，良率和效率控制是核心竞争力 |

### 先进封装布局

华天在 2.5D/3D 封装上技术水平不及长电和通富，但在 Fan-out 和 WLCSP 等中高端封装上产能充足、良率高。核心策略：**先在成熟先进封装上做到极致成本优势，再逐步向更高端技术延伸**。

### 投资逻辑与风险

| 逻辑 | 风险 |
|------|------|
| **国产替代弹性最大**：营收基数最小，国产化率每提升 1%，弹性最大 | **2.5D/3D 技术差距大**，短期难承接 AI 高端封装 |
| **成本护城河**：内陆建厂 + 规模化→毛利率优于同行 | **增速持续性存疑**，取决于国产化实际节奏 |
| **存储封装顺风**：HBM 需求暴增→存储封装需求，华天有工艺积累 | — |

## 四、其他值得关注的封装厂

### 晶方科技（603005.SH）

| 维度 | 信息 |
|------|------|
| **定位** | 全球 CIS（CMOS 图像传感器）封装龙头 |
| **核心业务** | WLCSP、Fan-out 封装，专注图像传感器/指纹识别芯片 |
| **客户** | 索尼、豪威科技（韦尔股份子公司）、格科微 |
| **看点** | 手机多摄像头 + 车载摄像头双驱动，CIS 封装需求持续扩张；晶圆级封装是公司核心技术壁垒 |
| **投资关联** | 消费电子复苏 + 汽车智能化 → CIS 出货量增加 → 晶方封装订单增加 |

### 气派科技（688216.SH）

| 维度 | 信息 |
|------|------|
| **定位** | 国内功率器件封装龙头 |
| **核心业务** | DIP/SOP/SOT 等传统封装 + 部分先进封装（QFN/DFN） |
| **客户** | 功率半导体公司（MOSFET/IGBT/SiC） |
| **看点** | 新能源车/光伏/储能拉动功率半导体需求，功率器件封装增长确定性强 |
| **投资关联** | 受益于 SiC 渗透率提升（SiC 封装难度远高于硅），在高端功率封装上有技术积累 |
