# 港股 — 先进封装公司分析

港股市场中，先进封装相关标的以 OSAT 封装厂为主，另有部分上游材料和设备公司通过子公司或关联公司在港股交易。

---

## 一、封装与测试

### 日月光投控（ASE Technology Holding / ASX）

> 日月光的台湾母公司（3711.TW）在台股交易，美股的 ADR（ASX）和港股无直接上市。但通过沪港通/深港通机制，内地投资者可通过港股通间接参与台股部分标的。日月光投控虽不在港股直接上市，但作为全球最大 OSAT，对 A 股封装厂的估值有风向标意义。

| 维度 | 信息 |
|------|------|
| **定位** | 全球最大 OSAT（独立封装测试厂），封装市占率 ~18%，全球排名第 2 |
| **商业模式** | 纯第三方来料封装——不绑定任何代工厂 |
| **封装平台** | VIPack（集成 Fan-out、2.5D、3D 等全系先进封装技术） |
| **核心客户** | 高通、联发科、苹果（部分）、英特尔（封装外包） |
| **投资关联（A 股）** | 日月光扩产→设备采购增加→利好北方华创等；技术路线迭代→为长电/通富指明追赶方向 |

### 华虹半导体（1347.HK）

| 维度 | 信息 |
|------|------|
| **定位** | 国内第二大晶圆代工厂，特色工艺（功率器件/模拟/嵌入式存储）代工龙头 |
| **先进封装关联** | 在特色工艺芯片的代工中，部分产品需要配套先进封装（如 SiP 模组）；华虹无锡基地有配套的先进封装产线规划 |
| **看点** | 与三安集成、中芯国际形成国内芯片制造+封装的组合；功率半导体需求（新能源车/光伏）拉动特色工艺芯片出货，间接拉动封装需求 |

## 二、材料与设备

### ASM Pacific Technology（0522.HK）

| 维度 | 信息 |
|------|------|
| **定位** | 全球半导体封装设备龙头，港股稀缺的半导体设备标的 |
| **核心产品** | 固晶机（Die Bonder）、引线键合机（Wire Bonder）、塑封设备——封装后道三大核心设备 |
| **先进封装关联** | 倒装焊（Flip Chip Bonder）是 FC-BGA/2.5D 封装的必需设备；热压键合（TCB）是 3D 封装中芯片堆叠的关键设备 |
| **竞争优势** | 固晶机市占率全球第一（~35%），技术和规模领先 |
| **看点** | 先进封装扩产直接拉动封装后道设备需求；公司在 TCB 设备上持续投入，有望在 3D 封装设备市场获得份额 |

### 建滔集团（0148.HK）

| 维度 | 信息 |
|------|------|
| **定位** | 全球最大覆铜板（CCL）和 PCB 制造商之一 |
| **先进封装关联** | 覆铜板是 PCB 和封装基板的基础材料；AI 芯片出货量暴增→高端 PCB/载板需求增加→覆铜板需求增加 |
| **看点** | 覆铜板是顺周期品种（芯片需求越好→覆铜板越旺），建滔作为全球龙头受益于半导体整体周期向上 |

## 三、港股投资要点

> 港股先进封装标的不如 A 股丰富，但 ASM Pacific 作为全球封装设备龙头的稀缺性突出。此外，日月光投控（台股）和台积电（台股）虽不直接在港股上市，但对 A 股封装板块的估值有重要参考价值。
