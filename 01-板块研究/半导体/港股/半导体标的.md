# 港股半导体 — 半导体标的

港股半导体标的分两类：一是与 A 股「A+H」两地上市的大陆晶圆厂（中芯国际、华虹半导体），二是总部在香港、业务全球化的设备公司（ASMPT）。对 A 股投资者而言，港股的参考价值在于**估值锚定**与**海外业务验证**。

---

## 一、中芯国际（00981.HK）

| 维度 | 信息 |
|------|------|
| **定位** | 与 A 股 688981 同一主体，港股 H 股上市 |
| **与 A 股关系** | A/H 两市同股同权，港股通常估值更低、流动性不同 |
| **2025 年报口径（港股，港元）** | 营收 727.5 亿港元（同比 <span style="color:red">+16.1%</span>）、归母净利 5.34 亿港元（同比 <span style="color:red">+39.0%</span>） |
| **2026Q1（港元）** | 营收 195.7 亿港元（同比 <span style="color:red">+11.5%</span>）、归母净利 15.4 亿港元（同比 <span style="color:red">+5.0%</span>） |
| **投资关联** | 港股提供「折价估值」参照；国际资金对大陆代工的风险定价更敏感，可作为情绪先行指标 |

> **关键逻辑**：中芯国际港股是观测「外资如何看待中国半导体自主化」的窗口。港股股价对地缘政策、出口管制消息反应通常比 A 股更剧烈，是风险偏好的温度计。

| 维度 | 信息 |

| **投资关联** | 特色工艺代工，港股估值锚；关注无锡新厂产能释放节奏 |

## 三、ASMPT（0522.HK）

| 维度 | 信息 |
|------|------|
| **定位** | 全球领先的后道封装设备 + 部分前道装配设备供应商，固晶机（Die Bond）全球第一 |
| **业务结构** | 半导体封装设备（含先进封装）+ SMT（表面贴装）设备 |
| **与大陆关联** | 先进封装扩产（CoWoS 外溢、OSAT 扩产）拉动其后道设备需求 |
| **投资关联** | 是先进封装设备方向的核心海外映射标的（详见先进封装板块港股文件）；前道业务占比相对小 |

> **映射关系**：ASMPT 的订单景气度可验证全球封装设备需求；其在先进封装设备上的进展，对 A 股封装设备（如快克智能等）有技术对标意义。
