# A股半导体 — 晶圆制造（Foundry）

晶圆制造是半导体 L1 基底层的「战略核心载体」——所有芯片设计（Fabless）最终都要在这里变成实物。A 股晶圆制造以中芯国际、华虹公司、晶合集成三家公司为代表，对应不同制程定位与客户结构。

---

## 一、中芯国际（688981.SH / 00981.HK）

| 维度 | 信息 |
|------|------|
| **定位** | 中国大陆晶圆代工绝对龙头，全球第四、大陆第一 |
| **制程覆盖** | 成熟制程（28nm 及以上）全面自主；部分先进制程（以 DUV 多重曝光推进，受 EUV 管制限制） |
| **2024 营收** | 577.96 亿元（同比 <span style="color:red">+27.72%</span>） |
| **2024 归母净利润** | 36.99 亿元（同比 <span style="color:green">-23.31%</span>） |
| **利润下滑原因** | 新厂（中芯京城/中芯深圳等）大规模转固，折旧压力上升；成熟制程价格竞争加剧 |
| **资本开支** | 持续高位（每年约 70-75 亿美元量级），逆周期扩产 |
| **客户结构** | 国内设计公司为主（手机 SoC、CIS、MCU、功率、射频），部分海外客户 |
| **投资关联** | 国产芯片自主化核心载体；产能利用率与 AI/成熟制程需求高度相关 |

> **关键逻辑**：中芯国际的营收增长（<span style="color:red">+27.72%</span>）说明量在快速放大，但利润下滑反映「以价换量 + 折旧高峰」的阵痛期。对投资者而言，它的核心价值不在短期利润，而在**战略不可替代性**——它是中国唯一能承接先进与成熟制程自主化任务的代工平台。

---

## 二、华虹公司（688347.SH / 01347.HK）

| 维度 | 信息 |
|------|------|
| **定位** | 特色工艺晶圆代工龙头（功率器件、MCU、CIS、IGBT、射频） |
| **制程覆盖** | 以 55nm-90nm 及特色工艺为主，专注「差异化」而非最先进线宽 |
| **2024 营收** | 143.88 亿元（同比 <span style="color:green">-11.36%</span>） |
| **2024 归母净利润** | 3.81 亿元（同比 <span style="color:green">-80.34%</span>） |
| **业绩下滑原因** | 功率/MCU 等特色工艺需求疲软、产能利用率低位、价格承压 |
| **投资关联** | 特色工艺自主化核心；无锡新厂（华虹九厂 12 英寸）投产是未来产能释放关键 |

> **关键逻辑**：华虹走「特色工艺」差异化路线，避开与台积电在先进制程的正面竞争。其 2024 业绩下滑主要是周期性的——特色工艺（功率/车规）处于去库存底部。一旦汽车电子、工业需求复苏，业绩弹性较大。

---

## 三、晶合集成（688249.SH）

| 维度 | 信息 |
|------|------|
| **定位** | LCD 显示驱动 IC（DDIC）晶圆代工龙头，向 CIS/MCU 等拓展 |
| **制程覆盖** | 以 55nm-150nm 显示驱动为主，特色工艺延伸 |
| **2024 营收** | 92.49 亿元（同比 <span style="color:red">+27.69%</span>） |
| **2024 归母净利润** | 5.33 亿元（同比 <span style="color:red">+151.78%</span>） |
| **高增长原因** | 显示驱动 IC 需求回暖 + 产能利用率提升 + 产品结构优化 |
| **投资关联** | 显示驱动代工细分龙头，受益面板产业链国产化 |

> **关键逻辑**：晶合集成是「细分赛道龙头 + 周期反转」的代表。其净利润 <span style="color:red">+151.78%</span> 的高增，主要来自显示驱动需求的周期性复苏，而非长期结构变化。投资需关注面板周期与产能扩张节奏。
