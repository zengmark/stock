# A股半导体 — 晶圆制造（Foundry）

晶圆制造是半导体 L1 基底层的「战略核心载体」——所有芯片设计（Fabless）最终都要在这里变成实物。A 股晶圆制造以中芯国际、华虹公司、晶合集成三家公司为代表，对应不同制程定位与客户结构。

---

## 一、中芯国际（688981.SH / 00981.HK）

| 维度 | 信息 |
|------|------|
| **定位** | 中国大陆晶圆代工绝对龙头，全球第四、大陆第一 |
| **制程覆盖** | 成熟制程（28nm 及以上）全面自主；部分先进制程（以 DUV 多重曝光推进，受 EUV 管制限制） |
| **2025 营收** | 671.5 亿元（同比 <span style="color:red">+16.1%</span>） |
| **2025 归母净利润** | 49.3 亿元（同比 <span style="color:red">+39.0%</span>） |
| **2026Q1** | 营收 180.4 亿元（同比 <span style="color:red">+11.5%</span>）、归母净利 14.2 亿元（同比 <span style="color:red">+5.0%</span>） |
| **利润变化** | 2024 年因新厂转固折旧高峰、利润下滑；2025 年净利恢复正增长，折旧压力逐步被放量收入消化 |
| **资本开支** | 持续高位（每年约 70-75 亿美元量级），逆周期扩产 |
| **客户结构** | 国内设计公司为主（手机 SoC、CIS、MCU、功率、射频），部分海外客户 |
| **投资关联** | 国产芯片自主化核心载体；产能利用率与 AI/成熟制程需求高度相关 |

> **关键逻辑**：中芯国际 2025 年营收稳健增长（<span style="color:red">+16.1%</span>），归母净利润恢复正增长（<span style="color:red">+39.0%</span>），结束了 2024 年利润下滑的阵痛期——新厂折旧高峰正逐步被放量收入消化。2026Q1 营收 180.4 亿元（<span style="color:red">+11.5%</span>）、净利 14.2 亿元（<span style="color:red">+5.0%</span>），延续稳健。对投资者而言，它的核心价值不在短期利润，而在**战略不可替代性**——它是中国唯一能承接先进与成熟制程自主化任务的代工平台。

## 二、华虹公司（688347.SH / 01347.HK）

| 维度 | 信息 |
|------|------|
| **定位** | 特色工艺晶圆代工龙头（功率器件、MCU、CIS、IGBT、射频） |
| **制程覆盖** | 以 55nm-90nm 及特色工艺为主，专注「差异化」而非最先进线宽 |
| **2025 营收** | 172.9 亿元（同比 <span style="color:red">+19.9%</span>） |
| **2025 归母净利润** | 3.95 亿元（同比 <span style="color:green">-5.6%</span>） |
| **2026Q1** | 营收 47.6 亿元（同比 <span style="color:red">+22.2%</span>）、归母净利 1.51 亿元（同比 <span style="color:red">+457%</span>） |
| **业绩特征** | 2025 营收恢复增长但净利仍小幅下滑（功率/车规等特色工艺尚在复苏途中）；2026Q1 净利高增确认周期反转 |
| **投资关联** | 特色工艺自主化核心；无锡新厂（华虹九厂 12 英寸）投产是未来产能释放关键 |

> **关键逻辑**：华虹走「特色工艺」差异化路线，避开与台积电在先进制程的正面竞争。其 2025 年营收恢复增长（<span style="color:red">+19.9%</span>）但净利仍小幅下滑（<span style="color:green">-5.6%</span>），主要是特色工艺（功率/车规）处于去库存后的复苏途中；而 2026Q1 归母净利润同比 <span style="color:red">+457%</span>，已清晰确认周期反转。无锡新厂产能释放将进一步打开成长空间。

## 三、晶合集成（688249.SH）

| 维度 | 信息 |
|------|------|
| **定位** | LCD 显示驱动 IC（DDIC）晶圆代工龙头，向 CIS/MCU 等拓展 |
| **制程覆盖** | 以 55nm-150nm 显示驱动为主，特色工艺延伸 |
| **2025 营收** | 108.9 亿元（同比 <span style="color:red">+17.7%</span>） |
| **2025 归母净利润** | 7.04 亿元（同比 <span style="color:red">+32.1%</span>） |
| **高增长原因** | 显示驱动 IC 需求回暖 + 产能利用率提升 + 产品结构优化 |
| **投资关联** | 显示驱动代工细分龙头，受益面板产业链国产化 |

> **关键逻辑**：晶合集成是「细分赛道龙头 + 周期反转」的代表。其 2025 年营收 <span style="color:red">+17.7%</span>、净利 <span style="color:red">+32.1%</span> 的持续增长，主要来自显示驱动需求的周期性复苏；但增速较 2024 年（+151.8%）明显回落，说明高基数下弹性减弱。投资需关注面板周期与产能扩张节奏。
