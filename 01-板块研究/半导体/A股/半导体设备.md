# A股半导体 — 半导体设备

半导体设备是 L1 基底层中**确定性最高**的环节——无论哪类芯片放量，晶圆厂都要先买设备、扩产线。A 股已形成一个覆盖八大类设备的「国产设备军团」，是国产替代最确定的方向。下文按平台型龙头与专精型公司分类。

---

## 一、平台型龙头

### 北方华创（002371.SZ）— 设备平台型绝对龙头

| 维度 | 信息 |
|------|------|
| **定位** | A 股半导体设备绝对龙头，平台型覆盖刻蚀、沉积（PVD/CVD/ALD）、清洗、热处理、电镀多环节 |
| **2025 营收** | 393.5 亿元（同比 <span style="color:red">+30.9%</span>） |
| **2025 归母净利润** | 55.2 亿元（同比 <span style="color:green">-1.8%</span>） |
| **2026Q1** | 营收 103.3 亿元（同比 <span style="color:red">+25.8%</span>）、归母净利 16.3 亿元（同比 <span style="color:red">+3.4%</span>） |
| **净利变化** | 2025 净利微降，主要因研发投入与股权激励摊销拖累；2026Q1 净利恢复正增长，盈利质量边际改善 |
| **投资关联** | 产品线最全、确定性最高；是晶圆厂扩产「卖铲子」逻辑的最大受益者 |

> **关键逻辑**：北方华创是唯一具备平台化能力的 A 股设备公司，单家即可满足晶圆厂大部分前道设备需求。其 2025 年营收维持 <span style="color:red">+30.9%</span> 高增，归母净利润 <span style="color:green">-1.8%</span>（研发投入与股权激励摊销阶段性拖累），2026Q1 净利恢复 <span style="color:red">+3.4%</span>。仍是半导体设备板块的「压舱石」。

## 二、刻蚀 / 沉积专精

| 维度 | 信息 |
|------|------|
| **定位** | 等离子体刻蚀（CCP）专精龙头，MOCVD 设备领先 |
| **2025 营收** | 123.9 亿元（同比 <span style="color:red">+36.6%</span>） |
| **2025 归母净利润** | 21.1 亿元（同比 <span style="color:red">+30.6%</span>） |
| **技术壁垒** | 刻蚀设备进入台积电 5nm 供应链，是国产设备中技术含量最高的代表之一 |

> **关键逻辑**：中微公司 2025 年营收 <span style="color:red">+36.6%</span>、净利 <span style="color:red">+30.6%</span>，前期高研发投入进入收获期，结束了 2024 年利润短期下滑的态势。刻蚀设备进入台积电 5nm，是国产替代「技术标杆」。
