# A股半导体 — 半导体材料

半导体材料是 L1 基底层中**国产替代空间最大**的环节——多数品类国产化率低于 30%，且具备「耗材逻辑」（每道工序重复消耗、客户认证后粘性极高）。下文按材料品类分组，覆盖硅片、CMP 材料、前驱体/特气、光刻胶、靶材等关键方向。

---

|------|------|
| **定位** | 国产 300mm（12 英寸）大硅片龙头，覆盖抛光片/外延片 |
| **2024 营收** | 33.88 亿元（同比 <span style="color:red">+6.18%</span>） |
| **2024 归母净利润** | -9.71 亿元（亏损，同比 <span style="color:green">-620.28%</span>） |
