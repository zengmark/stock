# A 股 AI 算力芯片 — 设计四强

A 股能规模化出货 AI 算力芯片的公司只有两家（寒武纪、海光），另外两家（景嘉微、龙芯）更偏「主题跟涨」。本文件按「纯 AI 芯片 → 双芯平台 → 军用 GPU → 自研 CPU」四类梳理。

> 数据来源：neodata-financial-search（东方财富）。A 股财务数据为 2025 年报，部分含 2026Q1。

---

## 一、寒武纪（688256.SH）— A 股纯 AI 芯片第一股

| 维度 | 信息 |
|------|------|
| **定位** | A 股唯一专注 AI 算力芯片的公司，国产替代弹性最大 |
| **上市板块** | 科创板 |
| **2025 营收** | ¥64.97 亿（同比 **+453.20%**） |
| **2025 归母净利润** | ¥20.59 亿（同比 +555%，**首次年度盈利**） |
| **2026Q1** | 营收 ¥28.85 亿（+159.56%），归母净利润 ¥10.13 亿（+185.04%） |
| **毛利率** | 55.15%（2025），54.33%（26Q1） |

### 产品矩阵

| 产品 | 定位 | 状态 |
|------|------|------|
| 思元 290/370/590 | 云端训练/推理芯片 | 主力出货，云端产品线占营收 99.69% |
| 新一代微架构 | 针对大模型推理/训练优化 | 研发中，提升编程灵活性、性能、功耗 |
| NeuWare 软件平台 | 兼容 PyTorch/Triton | 2026 年实现对 DeepSeek-V4、GLM-5 的 **Day0 级适配** |

### 客户与需求验证

- **合同负债爆发**：2025 年末 ¥0.01 亿 → 2026Q1 末 ¥3.96 亿（环比 +¥3.95 亿），验证下游需求高景气
- **备货充足**：2026Q1 末存货 ¥44.97 亿，预付款 ¥18.97 亿（环比 +155%），积极备货应对订单
- **客户结构**：运营商集采、政务信创、互联网厂商（国产替代刚需）

### 竞争壁垒

1. **国产最大第三方 AI 芯片供应商**：据 IDC，2025 年中国 AI 芯片国产化率从 30%→41%，寒武纪是最大的第三方供应商（不含华为/平头哥/昆仑芯等自用型）
2. **Day0 适配能力**：新模型发布当天就能跑，软件迁移成本极低——这是抢互联网客户的关键
3. **基数小、弹性大**：从 2024 年亏损 ¥4.5 亿到 2025 年盈利 ¥20.6 亿，业绩弹性 A 股第一

### 风险点

| 风险 | 影响程度 | 说明 |
|------|---------|------|
| 制程受限 | ★★★★ | 受光刻机限制，单卡性能落后英伟达，靠集群补 |
| 估值极高 | ★★★★ | 2026E PE 超 100 倍，市场已 price in 高增长 |
| 客户集中 | ★★★ | 政务/运营商占比高，订单节奏波动大 |
| 现金流承压 | ★★ | 2025 经营现金流为负（-¥4.98 亿），备货压力大 |

### 机构预测

多家机构维持「买入」，预计 2026-2028 年营收 ¥146-165 亿 / ¥237-284 亿 / ¥367-435 亿，归母净利润 ¥51 / ¥84 / ¥137 亿。高盛目标价 ¥2,406（对应 102 倍 2027E PE）。

## 二、海光信息（688041.SH）— CPU+DCU 双芯平台

| 维度 | 信息 |
|------|------|
| **定位** | 国内唯一同时具备 CPU 和 AI 协处理器（DCU）量产能力的公司 |
| **上市板块** | 科创板 |
| **2025 营收** | ¥143.77 亿（同比 **+56.92%**） |
| **2025 归母净利润** | ¥25.45 亿（同比 +31.79%） |
| **2026Q1** | 营收 ¥40.34 亿（+68.06%），归母净利润 ¥6.87 亿（+35.82%） |
| **毛利率** | 57.83%（2025），55.60%（26Q1，DCU 占比提升致毛利略降） |

### 双芯产品矩阵

| 产品线 | 定位 | 亮点 |
|--------|------|------|
| **海光 CPU**（3000/5000/7000 系列） | 兼容 x86 指令集的国产处理器 | 国家信创核心供应商，党政/金融/电信渗透率持续上升 |
| **海光 DCU**（深算系列） | 基于 GPGPU 架构，兼容类 CUDA | 深算三号已出货，深算四号研发中；适配 365 款主流大模型（覆盖 99% 非闭源模型） |
| **HSL 互联协议** | 2025 年开放 CPU 互联总线 | 让 xPU/IO/OS/OEM 厂商紧耦合互联，强化 CPU 算力底座地位 |
| **scaleX40 超节点** | 单节点 40 张 GPU，FP8 算力 28 PFLOPS | 国产智算基础设施突破，HBM 显存超 5TB |

### 客户与生态

- **信创核心**：国家税务总局、海关总署、国有银行、三大运营商均为客户
- **互联网突破**：与国内头部互联网厂商建立深度合作，定制化服务
- **落地广度**：DCU 已在 20+ 关键行业、300+ 应用场景落地
- **生态组织**：「光合组织」凝聚 6000+ 生态伙伴，15000+ 联合解决方案

### 竞争壁垒

1. **双芯唯一性**：A 股唯一「CPU + DCU」双产品线，异构计算协同优势
2. **x86 生态兼容**：CPU 兼容国际主流 x86，迁移成本极低，信创首选
3. **DCU 类 CUDA**：DTK 软件栈兼容 CUDA 环境，互联网客户低成本切换
4. **产销率验证**：2025 年高端处理器产销率 98.84%（同比 +23.49pct），销售规模显著扩张

### 风险点

| 风险 | 影响程度 | 说明 |
|------|---------|------|
| 毛利率下行 | ★★★ | DCU 占比提升拉低毛利（57.8%→55.6%） |
| 制程限制 | ★★★★ | 同寒武纪，受光刻机限制 |
| 估值高 | ★★★ | 2026E PE ~115 倍（机构预测） |
| 股份支付 | ★★ | 26Q1 股份支付影响 ¥2.4 亿，压制利润增速 |

### 机构预测

机构预计 2026-2028 年营收 ¥206-223 亿 / ¥280-302 亿 / ¥378-380 亿，归母净利润 ¥43-47 亿 / ¥61-67 亿 / ¥77-93 亿。维持「买入/推荐」评级。

## 三、景嘉微（300474.SZ）— 军用 GPU，民用 AI 待破局

| 维度 | 信息 |
|------|------|
| **定位** | 国产 GPU（军用起家），向民用 AI 扩展 |
| **上市板块** | 创业板 |
| **2025 营收** | ¥7.20 亿（同比 +54%） |
| **2025 归母净利润** | -¥1.65 亿（连续两年亏损） |
| **研发费用** | ¥4.28 亿（占营收 59%，研发投入大但产出慢） |

### 业务结构

| 产品 | 状态 |
|------|------|
| JM 系列 GPU | 军用图形显控主导，民用拓展缓慢 |
| 民用 GPU | 在研，AI 算力芯片尚未规模化出货 |

### 投资逻辑与风险

- **逻辑**：国产 GPU 主题龙头，市场情绪好时跟涨寒武纪/海光
- **现实**：业绩持续亏损，AI 芯片商业化进度落后，基本面支撑弱
- **定位**：主题跟涨标的，非基本面驱动

| 风险 | 说明 |
|------|------|
| 持续亏损 | 2024-2025 连续亏损，民用 AI 未兑现 |
| 研发转化慢 | 研发费用率高但产品落地慢 |
| 军品波动 | 军用订单节奏不规律 |

## 四、龙芯中科（688047.SH）— 自研架构 CPU，非典型 AI 标的

| 维度 | 信息 |
|------|------|
| **定位** | 自主 LoongArch 指令集 CPU，信创基础算力 |
| **上市板块** | 科创板 |
| **2025 营收** | ¥6.35 亿（同比 +25.99%） |
| **2025 归母净利润** | -¥4.55 亿（亏损收窄 27.22%） |
| **毛利率** | 47.06% |

### 业务结构

| 产品 | 状态 |
|------|------|
| LoongArch CPU | 自主指令集，不依赖 x86/ARM，安全性最高 |
| 服务器/工控 CPU | 信创政务场景 |

### 投资逻辑与风险

- **逻辑**：完全自主架构，信创「最底层」算力，政策驱动
- **现实**：① CPU 而非 AI 芯片，AI 算力属性弱；② 生态独立导致软件适配难，规模小；③ 持续亏损
- **定位**：信创主题标的，与 AI 算力主线相关性弱

| 风险 | 说明 |
|------|------|
| 非 AI 芯片 | 主营 CPU，AI 属性弱，跟涨逻辑牵强 |
| 生态独立 | 不兼容 x86/ARM，软件适配难，规模受限 |
| 持续亏损 | 2022 起连续亏损 |

## 五、A 股四强对比总览

| 维度 | 寒武纪 | 海光信息 | 景嘉微 | 龙芯中科 |
|------|--------|---------|--------|---------|
| **AI 属性** | ★★★★★ 纯 AI 芯片 | ★★★★★ DCU + CPU | ★★★ GPU 待破局 | ★★ CPU 非 AI |
| **2025 营收** | ¥65 亿（+453%） | ¥144 亿（+57%） | ¥7.2 亿（+54%） | ¥6.4 亿（+26%） |
| **2025 净利** | ¥20.6 亿（扭亏） | ¥25.5 亿（+32%） | -¥1.65 亿 | -¥4.55 亿 |
| **盈利能力** | ✅ 已盈利 | ✅ 已盈利 | ❌ 亏损 | ❌ 亏损 |
| **国产替代弹性** | 最大 | 大 | 中 | 小 |
| **投资定位** | 核心标的 | 核心标的 | 主题跟涨 | 信创主题 |

> **核心结论**：A 股投资 AI 算力芯片，**就是寒武纪 + 海光信息**。两家都已规模化盈利，是国产替代唯二的基本面标的。景嘉微、龙芯中科更多是主题跟涨，需注意业绩兑现风险。
