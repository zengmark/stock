# 美股 AI 算力芯片 — GPU 双雄 + ASIC + HBM

美股是 AI 算力芯片的主战场，全球顶级玩家几乎都在这里。本文件覆盖五家核心公司：英伟达（GPU 霸主）、AMD（追兵）、博通（ASIC 定制王者）、Marvell（ASIC 二线）、美光（HBM 弹性最大）。

> 数据来源：neodata-financial-search（东方财富）。美股财年与自然年不同，下文标注财年截止日。

---

## 一、英伟达（NVDA）— AI 时代的绝对霸主

| 维度 | 信息 |
|------|------|
| **定位** | AI GPU 全球霸主，市占率 ~80%，整条产业链的「心脏」 |
| **上市市场** | 纳斯达克 |
| **FY2026 营收**（截至 2026.1.25） | **$2159.38 亿**（同比 +65%） |
| **FY2026 净利润** | **$1200.67 亿**（同比 +65%） |
| **毛利率** | 71.1% |
| **EPS** | $4.90 |
| **FY2027Q1**（截至 2026.4.26） | 营收 **$816.15 亿**（同比 **+85%**），净利 **$583.21 亿**（同比 **+211%**） |

### 业务构成（FY2026 年报主营构成）

| 板块 | 营收 | 占比 |
|------|------|------|
| Compute & Networking（含数据中心 GPU / 网络） | $1934.79 亿 | 89.60% |
| Graphics（游戏 / 专业可视化 / 汽车等） | $224.59 亿 | 10.40% |

> neodata 年报主营构成仅聚合为上述两大类；英伟达官方披露的细分板块（数据中心 / 游戏 / 专业可视化 / 汽车 / OEM）占比数据未收录。

### 财务爆发力

英伟达的财报就是 AI 时代的「温度计」：

| 财年 | 营收 | 净利润 | 关键产品 |
|------|------|--------|---------|
| FY2023（截至 2023.1） | $269.74 亿 | $43.68 亿 | H100 即将放量 |
| FY2024 | $609.22 亿 | $297.60 亿 | H100 大卖，AI 爆发 |
| FY2025 | $1304.97 亿 | $728.80 亿 | H200 + B200 导入 |
| **FY2026** | **$2159.38 亿** | **$1200.67 亿** | B200 放量，Rubin 路上 |

> **3 年营收 8 倍**：从 FY2023 的 $270 亿到 FY2026 的 $2159 亿，这是人类商业史上最快的增长之一。毛利率 71%、净利率 56%——这是「设计 + 生态垄断」的极致变现。

### 产品 Roadmap（全产业的节奏表）

| 架构 | 产品 | 量产时间 | 核心升级 |
|------|------|---------|---------|
| Hopper | H100/H200 | 2022-2024 | HBM3/3e，AI 训练主力 |
| Blackwell | B200/B300 | 2024-2025 | 双 die 设计，FP4，算力翻倍 |
| Rubin | R100 | 2026E | HBM4 配套，下一代 |
| Rubin Ultra | — | 2027E | 进一步扩展 |

### 护城河

1. **CUDA 生态**（400 万开发者，18 年积累）——无法撼动
2. **硬件领先**（每代领先 AMD 1-2 年）
3. **客户深度绑定**（与云厂商联合定义产品）
4. **roadmap 节奏**（一年一代，对手追不上）

### 风险

| 风险 | 说明 |
|------|------|
| 估值高 | 市值超 $3 万亿，已 price in 高增长 |
| 客户自研 | 云厂商（Google/亚马逊）自研 ASIC 分流推理份额 |
| 出口管制 | 高端 GPU 卖不进中国，损失部分市场 |
| 周期风险 | AI 投资若放缓，高估值承压 |

## 二、AMD（AMD）— 唯一有威胁的追兵

| 维度 | 信息 |
|------|------|
| **定位** | AI GPU 二供，英伟达唯一能追的对手 |
| **上市市场** | 纳斯达克 |
| **FY2025 营收**（截至 2025.12.27） | **$346.39 亿**（同比 +34%） |
| **FY2025 净利润** | $42.69 亿（同比 +160%） |
| **毛利率** | 46.0% |
| **FY2026Q1**（截至 2026.3.28） | 营收 **$102.53 亿**（同比 **+38%**），净利 **$13.72 亿**（同比 **+94%**） |

### 业务构成（FY2025 年报主营构成）

| 板块 | 营收 | 占比 |
|------|------|------|
| Data Center（数据中心 CPU + GPU） | $166.35 亿 | 48.02% |
| Client and Gaming（客户端 + 游戏） | $145.50 亿 | 42.00% |
| Embedded（嵌入式） | $34.54 亿 | 9.97% |

### 财务趋势

| 财年 | 营收 | 净利润 |
|------|------|--------|
| FY2023 | $226.80 亿 | $8.54 亿 |
| FY2024 | $257.85 亿 | $16.41 亿 |
| **FY2025** | **$346.39 亿** | **$42.69 亿** |

### 产品与策略

| 产品 | 定位 | 优势 |
|------|------|------|
| MI300X | AI 推理/训练 GPU | 192GB HBM3（容量大于英伟达），适合大模型推理 |
| MI325X/MI350 | 迭代产品 | 追赶英伟达 Blackwell |
| EPYC 服务器 CPU | 数据中心 CPU | 与英伟达竞争服务器 CPU 市场 |

**AMD 的天花板**：永远是「二供」，不是「取代」。毛利率 46% vs 英伟达 71%——这就是定价权差异。AMD 的机会在推理市场（大内存优势）和英伟达产能不足时的替代。

## 三、博通（AVGO）— 被低估的 ASIC 王者

| 维度 | 信息 |
|------|------|
| **定位** | AI 定制 ASIC 龙头，「卖铲子给挖金矿的人」 |
| **上市市场** | 纳斯达克 |
| **FY2025 营收**（截至 2025.11.2） | **$638.87 亿**（同比 +24%） |
| **FY2025 净利润** | **$231.26 亿**（同比 +275%） |
| **毛利率** | 64.8% |
| **FY2026Q2**（截至 2026.5.3） | 营收 **$221.87 亿**（同比 **+48%**），净利 **$93.10 亿**（同比 **+87%**） |

### 业务构成（FY2025 年报主营构成）

| 板块 | 营收 | 占比 |
|------|------|------|
| Semiconductor Solutions（半导体，含 AI ASIC） | $368.58 亿 | 57.69% |
| Infrastructure Software（基础设施软件，含 VMware） | $270.29 亿 | 42.31% |

### 财务爆发力

| 财年 | 营收 | 净利润 |
|------|------|--------|
| FY2023 | $358.19 亿 | $140.82 亿 |
| FY2024 | $515.74 亿 | $61.68 亿（VMware 收购整合期） |
| **FY2025** | **$638.87 亿** | **$231.26 亿** |

> FY2025 净利润从 $62 亿飙到 $231 亿，一是 AI ASIC 放量，二是 VMware 收购整合完成释放利润。

### 商业模式：AI 时代的「定制代工」

博通不卖标准 GPU，而是**帮云巨头设计定制 ASIC**：

| 客户 | 合作内容 |
|------|---------|
| Google | TPU 的核心设计伙伴（已合作十余年） |
| Meta | 定制推理 ASIC |
| 字节跳动 | 定制 AI 芯片 |
| OpenAI | 传正在合作定制芯片 |

**为什么博通是隐藏赢家**：不管谁的 ASIC 卖得好，博通都赚设计费 + 每颗芯片抽成。2024 年 AI ASIC 营收 ~$120 亿，2025 年翻倍。当推理市场爆发、ASIC 渗透率提升，博通是最大受益者。

### 护城河

1. **SerDes/网络 IP**（高速互联技术，自研）
2. **定制 ASIC 经验**（与 Google 合作十余年，know-how 沉淀）
3. **客户关系**（顶级云厂商都是客户）

| 维度 | 信息 |
|------|------|
| **定位** | 定制 ASIC 二线厂商，博通的追赶者 |
| **上市市场** | 纳斯达克 |
| **FY2026 营收**（截至 2026.1.31） | **$81.95 亿**（同比 +42%） |
| **FY2026 净利润** | $26.70 亿（扭亏为盈） |
| **毛利率** | 51.0% |
| **FY2027Q1**（截至 2026.5.2） | 营收 **$24.18 亿**，净利 **$0.345 亿**（环比 FY2026 大幅回落，需关注 ASIC 需求节奏） |

### 业务构成（FY2026 年报主营构成）

| 板块 | 营收 | 占比 |
|------|------|------|
| Data Center（数据中心 ASIC） | $61.00 亿 | 74.44% |
| Communications and Other（通信及其他） | $20.94 亿 | 25.56% |

### 财务拐点

| 财年 | 营收 | 净利润 |
|------|------|--------|
| FY2024 | $55.08 亿 | -$9.33 亿（亏损） |
| FY2025 | $57.67 亿 | -$8.85 亿（亏损） |
| **FY2026** | **$81.95 亿** | **$26.70 亿（扭亏）** |

> Marvell 从连续两年亏损到 FY2026 扭亏，AI 定制 ASIC（亚马逊 Trainium 等）放量是核心驱动。弹性大但确定性弱于博通。

### 客户

- 亚马逊（Trainium 定制 ASIC）
- 其他云厂商（部分）

**定位**：博通的「替补」，当博通产能/接单饱和时承接。弹性大，但客户集中度高、确定性弱。

## 五、美光（MU）— HBM 弹性最大的存储巨头

| 维度 | 信息 |
|------|------|
| **定位** | HBM 三寡头之一，存储周期反转 + HBM 放量双击 |
| **上市市场** | 纳斯达克 |
| **FY2025 营收**（截至 2025.8.28） | **$373.78 亿**（同比 +49%） |
| **FY2025 净利润** | **$85.39 亿**（同比 +997%） |
| **毛利率** | 40.2% |
| **FY2026Q3**（截至 2026.5.28） | 营收 **$414.56 亿**（同比 **+346%**），净利 **$282.43 亿**（同比 **+1399%**） |

### 业务构成（FY2025 年报主营构成）

| 板块 | 营收 | 占比 |
|------|------|------|
| Cloud Memory Business Unit（云内存） | $135.24 亿 | 36.18% |
| Mobile and Client Business Unit（移动与客户端） | $118.59 亿 | 31.73% |
| Core Data Center Business Unit（核心数据中心） | $72.29 亿 | 19.34% |
| Automotive and Embedded Business Unit（汽车与嵌入式） | $47.53 亿 | 12.72% |
| All Other | $0.13 亿 | 0.03% |

### 存储周期的 V 型反转

| 财年 | 营收 | 净利润 | 关键事件 |
|------|------|--------|---------|
| FY2023 | $155.40 亿 | **-$58.33 亿（巨亏）** | 存储周期底部 |
| FY2024 | $251.11 亿 | $7.78 亿 | 复苏 |
| **FY2025** | **$373.78 亿** | **$85.39 亿** | HBM 放量，暴赚 |

> 从巨亏 $58 亿到暴赚 $85 亿，只用两年。HBM 供需紧张推动存储芯片量价齐升，美光是三家中产能扩张最激进的。

### HBM 份额提升逻辑

| 维度 | 美光的优势 |
|------|-----------|
| 份额 | 从 ~15% 向 ~25% 提升（扩张最激进） |
| 技术 | HBM3E 量产最晚但良率追赶快 |
| 客户 | 英伟达份额提升中（从二供向主力） |
| 弹性 | 三家中利润弹性最大（基数低） |

### 风险

| 风险 | 说明 |
|------|------|
| 存储周期 | HBM 供需平衡后，存储周期可能再次下行 |
| 份额竞争 | SK 海力士、三星技术领先，美光份额提升有天花板 |
| 资本开支 | HBM 扩产需要巨额资本开支，压制自由现金流 |
