# 02 产业链深度拆解

> **给投资者的第一句话**：CPO/硅光的产业链，上游是「光与电的原材料和芯片」，中游是「光引擎与光模块」，下游是「交换机/ASIC 与云厂商」。A股强在中游制造，美股强在上游布局与整机生态。

## 一、产业链全景（上中下游）

```
上游（材料/芯片）          中游（光引擎/模块）          下游（系统/客户）
─────────────────        ─────────────────        ─────────────────
激光器芯片(VCSEL/DFB)  →  硅光光引擎  →  可插拔/CPO  →  交换机/AI 加速卡
  源杰/Coherent/Lumentum      光器件(插芯/连接器)        中际旭创/新易盛/博通
                      →  光模块                   →  云厂商(英伟达/谷歌/微软)
磷化铟/硅光衬底              天孚/太辰光/光迅
  光迅/Coherent          ── 交换芯片(指挥中枢) ──
                          盛科/博通/Arista
```

## 二、逐环节拆解

### 1. 上游：激光器芯片 + 光材料（国产替代要塞）

- **激光器芯片（VCSEL/DFB/EML）**：硅光光源，决定光信号质量。源杰科技（A股）主攻 VCSEL/DFB；美股 Coherent（InP 衬底）、Lumentum（激光器）卡位上游。
- **磷化铟（InP）/硅光衬底**：光器件的「晶圆」。Coherent 是 InP 龙头；光迅科技纵向整合光芯片，降低对外依赖。
- **投资要点**：上游技术壁垒最高、国产替代空间大，但单体公司营收规模小（源杰 6 亿级），波动大。

### 2. 中游：光器件 + 光模块（A股最强环节）

- **光器件（插芯/连接器/无源组件）**：天孚通信（陶瓷插芯+无源/有源器件）、太辰光（MT 插芯/光纤连接器）是 CPO 高密度互连的「耗材」，确定性高、受技术路线切换影响小。
- **光模块（可插拔/CPO）**：中际旭创、新易盛为全球前二，直接受益 800G→1.6T 放量；华工科技、剑桥科技、长芯博创、联特科技、德科立为第二梯队。光模块是「业绩兑现最快、但预期波动最大」的环节。
- **光芯片+模块一体化**：光迅科技纵向整合，抗风险能力强。

### 3. 下游：交换芯片 + 系统（CPO 的「大脑」与「出口」）

- **交换芯片**：CPO 机柜级光互联的指挥中枢，决定光互联的带宽与拓扑。博通（Tomahawk）、Arista（系统）、盛科通信（A股国产）卡位；盛科短期亏损换卡位。
- **系统/客户**：交换机（Arista/思科）、AI 加速卡（英伟达）、云厂商（谷歌/微软/Meta）——它们是需求源头，也是 CPO 量产的「开关」。

## 三、谁卡位、谁受益（投资者视角）

| 环节 | 卡位方 | 投资属性 |
|------|--------|---------|
| 激光器芯片 | 源杰、Coherent、Lumentum | 高弹性、国产替代 |
| 光材料(InP) | Coherent、光迅 | 上游卖铲人 |
| 光器件 | 天孚、太辰光 | 确定性高（耗材） |
| 光模块 | 中际旭创、新易盛、华工等 | 业绩弹性+预期波动 |
| 交换芯片 | 博通、盛科 | 长期确定性、短期承压 |

## 四、与光模块板块的关系

本板块（CPO/硅光）与「光模块」板块同源：光模块是 CPO 的「现在形态」，CPO 是光模块的「未来演进」。光模块板块覆盖可插拔主力（800G/1.6T）的业绩与格局；本板块聚焦「电退光进」的演进工艺（硅光/CPO）与机柜级光互联（交换芯片），两者互补、不重复。
