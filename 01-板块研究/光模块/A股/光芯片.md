# A股光模块 — 光芯片三剑客

光芯片是光模块产业链中「国产替代空间最大、技术壁垒最高、弹性最大」的环节。当前高端 EML 芯片国产化率 <5%，CW 光源 ~15%，VCSEL ~30%。如果把光模块比作汽车，光芯片就是发动机。

---

## 一、源杰科技（688498.SH）— 国产光芯片第一股

| 维度 | 信息 |
|------|------|
| **定位** | A 股最纯正的光芯片公司，专注激光器芯片（发射端） |
| **上市板块** | 上交所科创板 |
| **2025 营收** | ¥6.01 亿（2025 年报），归母净利 1.91 亿 |
| **2026Q1** | 营收 3.55 亿，同比 <span style="color:#d90000">+320.94%</span>，归母净利 1.79 亿（同比 <span style="color:#d90000">+1153%</span>，毛利率升至 77.81%） |
| **产品型谱** | CW 光源 → DFB → EML，从易到难阶梯布局 |

### 产品矩阵与进度

| 产品 | 速率 | 应用场景 | 进度 | 国产替代空间 |
|------|------|---------|------|------------|
| **CW 光源** | 70mW+ | 硅光模块核心光源 | ✅ 批量出货 | 海外缺口 25-30%，国产替代紧迫 |
| **25G/50G DFB** | 25G/50G/通道 | 100G/400G 光模块 | ✅ 已量产 | 国产化率 ~20% |
| **100G DFB** | 100G/通道 | 高速光模块 | 🔶 送样中 | 国产化率 <10% |
| **100G EML** | 100G/通道 | 800G 光模块 | 🔶 小批量出货 | 国产化率 <5% |
| **200G EML** | 200G/通道 | 1.6T 光模块 | 🔶 送样验证 | 国产化率 ≈0% |

### 核心投资逻辑：EML 验证是「0→1」拐点

源杰科技当前估值的核心分歧在于：CW 光源（已量产）提供了当前的基本盘，但市场给的是 EML 通过验证后的远期估值。

| 情景 | 股价影响 | 概率判断 |
|------|---------|---------|
| **100G/200G EML 通过大客户验证并规模放量** | 股价翻倍级弹性 | 2026 下半年有望兑现（100G EML 已小批量出货） |
| **EML 规模放量不及预期或验证反复** | 估值回归 CW 基本盘，大幅回调 | 不可忽视的风险 |
| **200G EML 率先规模上量适配 1.6T** | 估值跃迁为平台型光芯片公司 | 远期，2-3 年维度 |

> **投资者核心关注**：时刻跟踪源杰科技的 EML 放量进展。100G EML 已小批量出货、200G EML 送样中，一旦规模放量并公告大客户，是从「CW 光源为主」到「CW + EML 全序列光芯片平台」的质变——这个拐点对标的高价值光芯片（EML）市场规模会显著扩大。

### 竞争壁垒

- **外延片生长能力**：EML 芯片的外延层结构极复杂（几十至上百层原子级精度），全球只有少数公司掌握
- **认证壁垒**：光芯片通过客户验证需要 12-24 个月，进入供应链后替换成本极高
- **人才稀缺**：国内能做高速光芯片设计的团队屈指可数

### 风险点

- EML 验证结果和时间不确定（最大风险）
- 估值较高，已包含 EML 成功的部分预期
- CW 光源单价低于高端 EML，长期成长弹性仍依赖 EML 上量打开（2026Q1 CW 放量已验证短期高增长与高毛利，但可持续性需持续跟踪）

## 二、长光华芯（688048.SH）— 激光芯片龙头 + 光通信全系列（含 EML）

| 维度 | 信息 |
|------|------|
| **定位** | 高功率半导体激光芯片国内龙头，同时布局 **EML / VCSEL / DFB / PIN 全系列高速光通信芯片** |
| **上市板块** | 上交所科创板 |
| **2025 营收** | ¥4.77 亿（2025 年报），归母净利 0.22 亿（全年扭亏为盈） |
| **2026Q1** | 营收 1.30 亿，同比 <span style="color:#d90000">+37.81%</span>，同比扭亏为盈 |

> **重要更正认知**：长光华芯并非「只做 VCSEL / 高功率激光」——它同样是 **EML 玩家**，且进度不弱于源杰。2025 年 11 月公司公告：**100G EML 已实现量产出货，200G EML 已进入客户送样阶段**；其 200G PAM4 EML 适配 800G/1.6T 硅光模块，8 通道 100G EML 可配套 800G FR8 光模块。光通信芯片已获海外大厂验证，下一代硅光产线预计 2027 年投产。

### 业务结构

| 业务 | 定位 | 市场地位 | 光模块关联 |
|------|------|---------|-----------|
| **高功率激光芯片** | 当前营收基本盘 | 国内龙头 | 无直接关联 |
| **光通信芯片（EML/DFB/VCSEL/PIN）** | 新增长极，快速放量中 | 全系列布局，100G EML 已量产、200G EML 送样 | **直接**（AI 数据中心高速互联核心） |
| **激光雷达 / 3D 传感芯片** | 起步期 | — | 间接（自动驾驶 / 消费电子） |

> 注：当前营收仍以高功率激光芯片为主，光通信芯片处于「兑现初期 → 规模放量」的过渡阶段，是未来弹性的核心看点。

### 光通信关联分析

长光华芯与源杰科技在光通信芯片上**既有重叠也有差异**——两者都是 EML/DFB 玩家，是直接竞争者；差异在于收入结构和产品覆盖：

| 维度 | 长光华芯 | 源杰科技 |
|------|---------|---------|
| 光通信芯片覆盖 | EML / DFB / VCSEL / PIN **全系列** | EML / DFB / CW 光源（无 VCSEL/PIN） |
| EML 进度 | 100G EML **已量产出货**，200G EML 送样 | 100G EML **小批量出货**，200G EML 送样 |
| 当前收入主力 | 高功率激光芯片（光通信仍在兑现初期） | 光通信芯片（CW 光源 2026Q1 已放量爆发） |
| 与 AI 关联度 | 直接（EML 全系列切入 800G/1.6T） | 直接（CW 光源已放量 + EML 蓄势） |

> **投资者关键认知**：长光华芯是「激光芯片龙头 × 光通信全系列（含 EML）」的复合标的。它同样是纯正的 AI 光芯片玩家，EML 进度不弱于源杰；区别在于源杰当下已靠 CW 光源实现业绩爆发、报表兑现更快，而长光华芯的光通信收入尚在爬坡、当前营收仍靠高功率激光支撑，胜在有更稳的基本盘打底。看重业绩兑现确定性选源杰，看重「激光基本盘 + 光通信弹性」双轮驱动选长光。

### 风险点

- 光通信芯片仍处兑现初期，当前营收高度依赖高功率激光芯片，报表弹性释放晚于源杰
- 100G/200G EML 的规模放量节奏与客户验证结果存在不确定性
- 高功率激光业务与下游（锐科激光等）景气度高度相关

## 三、仕佳光子（688313.SH）— AWG + CW 光源双主业

| 维度 | 信息 |
|------|------|
| **定位** | PLC 分路器全球龙头，光通信 AWG 芯片 + CW 光源供应商 |
| **上市板块** | 上交所科创板 |
| **2025 营收** | ¥21.29 亿（2025 年报，同比 <span style="color:#d90000">+98.15%</span>），归母净利 3.72 亿 |
| **2026Q1** | 营收 5.77 亿，同比 <span style="color:#d90000">+32.18%</span>，归母净利 1.16 亿 |

### 业务结构

| 业务 | 市场地位 | 驱动逻辑 |
|------|---------|---------|
| **PLC 分路器** | 全球龙头 | 光纤到户（FTTH），成熟基建逻辑 |
| **AWG 芯片** | 国产前列 | 波分复用（WDM），5G/数据中心 |
| **CW 光源** | 国产前三 | AI 硅光模块拉动 |

### 投资逻辑

仕佳光子是光芯片板块的「稳健入门」标的：

| 优势 | 说明 |
|------|------|
| **PLC/AWG 基本盘** | 全球光纤到户和 5G 网络建设提供稳定收入 |
| **CW 光源弹性** | 硅光模块放量 → CW 光源需求暴增 → 仕佳光子出货受益 |
| **估值合理** | 相比源杰科技，仕佳光子的估值近期相对合理（PLC 业务提供安全边际） |

### 风险点

- CW 光源竞争激烈（源杰科技、鼎芯光电、海外巨头均有布局）
- EML 等高价值光芯片无布局，天花板较低
- PLC 业务增速趋缓（光纤到户建设高峰已过）
