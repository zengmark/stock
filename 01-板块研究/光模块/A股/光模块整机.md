# A股光模块 — 整机四强

A 股光模块整机制造是全球最具竞争力的环节。LightCounting 2025 年全球 Top 10 光模块供应商中，中国企业占 7 席，前两名均为中国公司。

---

## 一、中际旭创（300308.SZ）— 全球光模块之王

| 维度 | 信息 |
|------|------|
| **定位** | 全球光模块销售额第 1，LightCounting 2025 排名第 1 |
| **上市板块** | 深交所创业板 |
| **2025 营收（年报）** | 382.40 亿（人民币），同比 <span style="color:#d90000">+60.25%</span>；归母净利润 107.97 亿，同比 <span style="color:#d90000">+108.78%</span> |
| **2026Q1** | 营收 194.96 亿，同比 <span style="color:#d90000">+192%</span>；归母净利润 57.35 亿 |

### 产品矩阵

| 产品 | 状态 | 市场地位 |
|------|------|---------|
| 800G 光模块 | 全球最大供应商 | 2025-2026 年主力出货，市占率第一 |
| 1.6T 光模块 | 已批量交付 | NVIDIA GB300 核心一供，2026E 出货 ~75 万只 |
| 3.2T 光模块 | 原型机送样 | 全球首家送样，技术代差领先约 1 年 |
| 硅光芯片 | 自研 | 800G/1.6T 硅光模块进度全球领先 |
| CPO 光模块 | 研发中 | 配合 NVIDIA、博通等客户开发 |

### 客户结构

- **NVIDIA**：最大客户，收入占比约 60%+，在 NVIDIA GB300 光模块采购中份额约 70%+。中际旭创长期担任 NVIDIA GPU 光模块一供，客户关系极深
- **Google**：第二大客户，800G 光模块持续供货
- **Meta、Amazon、Microsoft**：均有深度合作

### 竞争壁垒

1. **产线规模与自动化**：中际旭创的光模块产线自动化程度全球最高，良率和成本控制能力领先全行业
2. **NVIDIA 深度绑定**：与 NVIDIA 的长期合作形成了「联合研发→独家供应→持续迭代」的正循环，新进入者极难突破
3. **全栈能力**：从硅光芯片设计到模块封装到系统测试，全链路自研，垂直整合优势显著
4. **先发优势**：每一代新品首发都领先竞争对手 6-12 个月，持续吃第一波最高利润

### 风险点

| 风险 | 影响程度 | 说明 |
|------|---------|------|
| 客户集中度过高 | ★★★★ | NVIDIA 一家占 60%+，若其份额被新易盛等蚕食，冲击巨大 |
| 1.6T 订单波动 | ★★★ | GB300 出货节奏直接影响季度业绩 |
| 竞争加剧 | ★★★ | 新易盛 1.6T 良率极佳，正在缩小差距 |
| 光芯片紧缺 | ★★ | 海外 EML/CW 芯片供给紧张可能限制出货量 |

## 二、新易盛（300502.SZ）— 弹性最大的追兵

| 维度 | 信息 |
|------|------|
| **定位** | 全球第 2 光模块厂商（LightCounting 2025），中国第 2 |
| **上市板块** | 深交所创业板 |
| **2025 营收（年报）** | 248.42 亿（人民币），同比 <span style="color:#d90000">+187.29%</span>；归母净利润 95.32 亿，同比 <span style="color:#d90000">+235.89%</span> |
| **2026Q1** | 营收 83.40 亿，同比 <span style="color:#d90000">+105.8%</span>；归母净利润 27.80 亿，同比 <span style="color:#d90000">+76.8%</span> |

### 产品与竞争力

| 产品 | 状态 | 关键点 |
|------|------|--------|
| 800G 光模块 | 第二大供应商 | 2025 年放量，出货量快速增长 |
| 1.6T 光模块 | 批量交付 | 良率 ~92%（行业领先），NVIDIA+Meta 均通过验证 |
| CPO 光模块 | 布局中 | 1.6T CPO 方案验证中 |
| LPO 光模块 | **全球领先** | 首家 LPO 量产厂商，800G LPO 独供英伟达；LPO 差异化路线是核心竞争身位 |

### 核心逻辑：弹性标的

新易盛是光模块板块中弹性最大的标的，理由三条：

| 逻辑 | 说明 |
|------|------|
| **体量差** | 新易盛规模约中际旭创的 55%，同等营收增量带来的利润弹性更大 |
| **良率差异** | 新易盛 1.6T 良率高于行业平均，意味着同等出货下毛利率更高 |
| **追赶空间** | 中际旭创独占 NVIDIA 一供位置，新易盛从二供切入后份额爬坡是明确的增量 |

### 风险点

| 风险 | 影响程度 | 说明 |
|------|---------|------|
| 客户进展不确定 | ★★★ | 能否进一步扩大 NVIDIA/Meta 份额是关键 |
| 规模劣势 | ★★ | 与中际旭创的规模差距仍大，成本控制有劣势 |
| 产品迭代节奏 | ★★ | 3.2T 等下一代产品的布局速度不如中际旭创 |

## 三、华工科技（000988.SZ）— 光电器件 + 激光 + 传感器三业务

| 维度 | 信息 |
|------|------|
| **定位** | 中国光模块第二梯队龙头 + 激光设备龙头 + 传感器核心供应商，三大业务驱动 |
| **上市板块** | 深交所主板 |
| **2025 总营收** | 143.55 亿元（光电器件 / 激光装备 / 敏感元器件三大主业 + 其他），同比稳健增长 |
| **光电器件系列（含光模块）** | 60.97 亿（占比 42.5%；光模块为核心组成部分，同比约 <span style="color:#d90000">+125%</span>） |
| **敏感元器件（传感器）** | 40.27 亿（占比 28.0%）—— ⚠️ 此前「双引擎」框架漏写的重要业务线 |
| **激光加工装备** | 36.36 亿（占比 25.3%） |

### 光模块业务

| 产品 | 状态 | 关键点 |
|------|------|--------|
| 800G | 批量交付 | 海外头部云厂商客户覆盖 |
| 1.6T | 出海落地 | 已获海外头部云厂商验证通过并开始供货 |
| 光芯片 | 自研部分 | 激光芯片技术复用光通信 |

### 三大主业结构（合计占 ~96%，其余为其他）

| 维度 | 光电器件（含光模块） | 敏感元器件（传感器） | 激光加工装备 |
|------|--------|------|------|
| 2025 营收 | 60.97 亿（42.5%） | 40.27 亿（28.0%） | 36.36 亿（25.3%） |
| 驱动逻辑 | AI 算力基建（光模块） | 汽车电子 / 消费电子感知 | 高端制造升级 |
| 增速 | 快（光模块 >100%） | 稳 | 稳（20-30%） |
| 与光模块关联 | 直接（光模块是核心） | 间接 | 间接（激光复用光通信） |

> **业务结构更正**：华工科技并非「光模块 + 激光」双主业，而是**三大业务并列**——敏感元器件（传感器，40.27 亿、占 28%）是此前描述中被遗漏的重要板块。对投资者而言，华工的利润波动同时受三条线影响：光模块看 AI 资本开支、传感器看汽车 / 消费电子、激光看制造业景气。光模块提供弹性与估值溢价，传感器与激光提供现金流安全垫。

### 风险点

- 三大业务分散管理层精力
- 光模块体量远小於中际旭创/新易盛，拿大单的竞争力有限
- 1.6T 份额能否持续扩大待验证

## 四、剑桥科技（603083.SH）— 小而美弹性标的

| 维度 | 信息 |
|------|------|
| **定位** | 800G 光模块量产追赶者；总营收以电信宽带 / 高速光模块 / 无线边缘计算三块为主，光模块约占 35% |
| **上市板块** | 上交所主板 |
| **2025 总营收** | 48.22 亿元（同比稳健增长） |
| **其中高速光模块** | 16.75 亿元（占比 34.7%），其余为电信宽带（19.92 亿）/ 无线与边缘计算（11.54 亿） |

### 业务特点

| 项目 | 详情 |
|------|------|
| 核心产品 | 800G 光模块已量产，1.6T 研发中 |
| 核心客户 | 海外云厂商（规模较小） |
| 竞争模式 | 性价比路线，不与中际旭创/新易盛在头部客户正面竞争 |

### 投资逻辑

剑桥科技是光模块板块的「博弈型标的」——不是行业龙头，也不是国产替代逻辑，而是**小市值 × 高弹性**：

| 情景 | 预期影响 |
|------|---------|
| 拿到头部云厂商大订单 | 股价弹性极大（低基数效应） |
| 1.6T 通过客户验证 | 估值体系可能从「小厂」切换为「1.6T 新玩家」 |
| AI 情绪退潮 | 更小的体量和更弱的客户基础 → 抗跌性弱于头部 |

### 风险点

- 与头部公司的技术差距较大（中际旭创已送样 3.2T，剑桥刚上 800G）
- 客户基础薄弱，业绩弹性是「双刃剑」
- 市值小，流动性差
