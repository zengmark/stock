# 第四章：全市场核心公司分析

光模块产业链覆盖 A 股、美股、港股，其中 A 股是全球光模块制造的核心市场。本章按「整机模组 → 光器件 → 光芯片 → 交换机 → 美股对标」六个维度梳理。

---

## 一、A 股核心公司

A 股光模块标的最丰富，占据全球光模块制造前六中的四席。

| 环节 | 代表公司 | 投资逻辑 | 详细分析 |
|------|---------|---------|---------|
| **光模块整机** | 中际旭创、新易盛、华工科技、剑桥科技 | 直接受益于 AI 算力集群，确定性最强 | [→ 光模块整机](./A股/光模块整机.md) |
| **光器件/光引擎** | 天孚通信、光迅科技 | 卖铲子 + 耗材叠加逻辑，CPO 核心受益 | [→ 光器件](./A股/光器件.md) |
| **光芯片** | 源杰科技、长光华芯、仕佳光子 | 国产替代空间最大（<10%→33%+），弹性最高 | [→ 光芯片](./A股/光芯片.md) |
| **交换机** | 紫光股份、锐捷网络、菲菱科思 | 设备型需求，CPO 集成方逻辑 | [→ 交换机](./A股/交换机.md) |

| 优先级 | 方向 | 核心理由 |
|--------|------|---------|
| ★★★★★ | 光模块整机 | AI 算力最直接受益，确定性最强 |

## 二、A 股公司详细分析

> 以下为各公司简明分析卡片。完整深度分析（包含财务数据、估值、客户结构）请点击对应章节链接。

### 2.1 光模块整机四强

#### 中际旭创（300308.SZ）— 全球光模块之王

| 维度 | 信息 |
|------|------|
| **市场地位** | 2025 年全球光模块销售额第 1（LightCounting），市占率 28-30% |
| **产品矩阵** | 800G 全球最大供应商；1.6T 市占率 ~50%+，为 NVIDIA GB300 份额 70%+；3.2T 原型机已送样 |
| **核心客户** | NVIDIA（最大客户，收入占比约 60%+，GB300 光模块采购份额约 70%+）、Google、Meta、Amazon |
| **技术路线** | 自研硅光芯片 + 模块垂直整合，800G/1.6T 硅光进度领先 |
| **竞争优势** | 产线规模、自动化水平、良率控制全面领先，NVIDIA 一供地位稳固 |
| **2025 营收** | 382.40 亿（人民币，年报口径），同比 <span style="color:#d90000">+60.25%</span>；归母净利润 107.97 亿，同比 <span style="color:#d90000">+108.78%</span> |
| **风险点** | 客户集中度极高（NVIDIA 一家独大）；1.6T 订单波动可能带来季度业绩波动 |

#### 新易盛（300502.SZ）— 弹性最大的光模块标的

| 维度 | 信息 |
|------|------|
| **市场地位** | 2025 年全球第 2（超越 Coherent），预计 2026 保持第 2 |
| **产品矩阵** | 800G 第二大供应商；1.6T 良率 92%（行业领先）；CPO 光模块布局 |
| **核心客户** | NVIDIA、Meta 等 |
| **竞争优势** | 1.6T 良率最高（92% vs 行业 ~80-85%），毛利率弹性更大 |
| **2025 营收** | 248.42 亿（人民币，年报口径），同比 <span style="color:#d90000">+187.29%</span>；归母净利润 95.32 亿，同比 <span style="color:#d90000">+235.89%</span> |
| **弹性逻辑** | 相比中际旭创，新易盛规模小一倍以上，同等订单增长的利润弹性更大 |
| **风险点** | 1.6T 份额能否进一步扩大取决于良率优势能否持续保持 |

#### 华工科技（000988.SZ）— 光模块 + 激光双引擎

| 维度 | 信息 |
|------|------|
| **市场地位** | 2025 年全球第 5，中国光模块第二梯队龙头 |
| **光模块产品** | 800G 批量交付，1.6T 出海落地（海外头部云厂商客户） |
| **激光业务** | 高功率激光设备（切割/焊接/打标），国内龙头 |
| **2025 光模块营收** | ~$9 亿（约 65 亿人民币），同比 +125% |
| **双主线逻辑** | 光模块享受 AI 增长红利；激光享受高端制造升级红利 |
| **风险点** | 双主业分散资源，任一业务波动都会影响整体业绩 |

#### 剑桥科技（603083.SH）— 小而美的弹性标的

| 维度 | 信息 |
|------|------|
| **定位** | 800G 光模块量产追赶者，规模较小但增速潜力大 |
| **产品** | 800G 已量产，1.6T 研发中 |
| **特点** | 小市值 × 高增速弹性的博弈型标的，订单斩获决定股价弹性 |
| **风险点** | 与头部企业差距明显，业绩确定性不如中际旭创/新易盛 |

### 2.2 光器件 / 光引擎

#### 天孚通信（300394.SZ）— 光引擎全球「隐形冠军」

| 维度 | 信息 |
|------|------|
| **市场地位** | 全球光器件/光引擎龙头，几乎所有头部光模块厂都是其客户 |
| **核心产品** | 光引擎（Silicon Photonics 光引擎）、光器件（TOSA/ROSA、FA、MT 等）、精密陶瓷 |
| **CPO 关联** | CPO 需要高集成度的光引擎，天孚是全球光引擎产能最大的供应商 |
| **商业模式** | 「卖铲子」——不管哪家光模块厂商的模块卖得好，天孚都能受益 |
| **竞争优势** | 精密制造能力极深，光器件良率和一致性难以复制 |
| **2025 营收** | 51.63 亿（人民币，年报口径），同比 <span style="color:#d90000">+58.79%</span>；归母净利润 20.17 亿，同比 <span style="color:#d90000">+50.15%</span> |
| **风险点** | 增速可能随光模块行业增速趋缓而放缓，但 CPO 带来新增量 |

#### 光迅科技（002281.SZ）— 垂直整合最深的国资光通信龙头

| 维度 | 信息 |
|------|------|
| **市场地位** | 中国信科集团旗下，国内唯一实现「光芯片-光器件-光模块」全链条自给的公司 |
| **核心产品** | 光模块（800G/1.6T 均有）+ 光芯片（DFB/EML/CW/VCSEL 均有）+ 光器件 |
| **竞争优势** | 光芯片自给率国内最高，上游成本控制能力最强 |
| **2025 营收** | 119.29 亿（人民币，年报口径），同比 <span style="color:#d90000">+44.20%</span>；归母净利润 9.46 亿，同比 <span style="color:#d90000">+43.10%</span> |
| **风险点** | 国资背景，业绩弹性可能不如民企（中际旭创/新易盛） |

### 2.3 光芯片三剑客

#### 源杰科技（688498.SH）— 国产光芯片第一股

| 维度 | 信息 |
|------|------|
| **定位** | A 股最纯正的光芯片标的，专注激光器芯片（发射端） |
| **产品进度** | CW 70mW 光源 → 已放量爆发；25G/50G DFB → 已量产；100G DFB → 送样；100G EML → 小批量出货；200G EML → 送样验证 |
| **国产替代逻辑** | CW 光源已兑现业绩爆发；100G/200G EML 规模放量将从「CW 为主」升级为「CW + EML 全序列光芯片平台」|
| **风险点** | EML 规模放量节奏与客户验证不确定；估值较高（已含 EML 预期） |

#### 长光华芯（688048.SH）— 激光芯片龙头 + 光通信全系列（含 EML）

| 维度 | 信息 |
|------|------|
| **定位** | 高功率半导体激光芯片国内龙头，同时布局 EML/VCSEL/DFB/PIN 全系列光通信芯片 |
| **光通信关联** | **直接**：100G EML 已量产出货、200G EML 送样，200G PAM4 EML 适配 800G/1.6T 硅光模块，是源杰的直接竞争者 |
| **双赛道** | 高功率激光（当前营收基本盘）+ 光通信芯片（新增长极，快速放量中） |
| **风险点** | 光通信收入仍处兑现初期、当前营收依赖高功率激光；100G/200G EML 规模放量节奏存不确定性 |

#### 仕佳光子（688313.SH）— AWG + CW 光源双主业

| 维度 | 信息 |
|------|------|
| **定位** | PLC 分路器全球龙头（用于光纤到户），光通信 AWG 芯片国产替代 |
| **AI 关联** | CW 光源（DFB 激光器）用于硅光模块，已批量出货 |
| **基本面** | 传统 PLC/AWG 提供稳定收入，CW 光源提供 AI 弹性 |

### 2.4 交换机（共进股份）

#### 共进股份（603118.SH）— AI 算力代工弹性标的

| 维度 | 信息 |
|------|------|
| **定位** | 交换机 / AI 服务器代工龙头，受益于 AI 算力基建放量与交换机升级 |
| **上市板块** | 上交所主板 |
| **2025 营收** | 91.98 亿（人民币，年报口径），同比 <span style="color:#d90000">+9.81%</span>；归母净利润 0.78 亿，同比 <span style="color:#d90000">+197.29%</span> |
| **2026Q1** | 营收约 21.8 亿（同比 <span style="color:#d90000">+6.26%</span>），归母净利润约 0.23 亿（同比 <span style="color:#d90000">+9.65%</span>，按每股收益 0.03 元测算） |

> **逻辑**：交换机/服务器代工具备「出货量弹性」——AI 集群建设直接拉动代工订单，是光模块板块的「卖水人」延伸。

## 三、海外核心公司

海外公司覆盖美股、台股、日股，对 A 股投资有三大意义：**技术对标**、**景气度验证**、**供给瓶颈预警**。

| 公司 | 代码 | 市场 | 核心定位 |
|------|------|------|---------|
| **Coherent** | COHR | 美股 | 全球第 3 光模块厂商，光芯片+模块垂直整合 |
| **Lumentum** | LITE | 美股 | 光芯片（EML/VCSEL）巨头，NVIDIA 40 亿美金订单受益者 |
| **Marvell** | MRVL | 美股 | DSP 芯片双寡头（市占 ~60%），定制 ASIC + 光互联 |
| **Broadcom** | AVGO | 美股 | DSP + 交换机芯片 + 硅光芯片，全栈光互联 |
| **Credo** | CRDO | 美股 | 高速 SerDes/IP，AEC 有源电缆（DAC 替代） |
| **Arista** | ANET | 美股 | 数据中心交换机龙头，AI 集群组网核心 |
| **NVIDIA** | NVDA | 美股 | 光模块最大买家，CPO 交换机推动者 |
| **富士通光器件** | 6702.T | 日股 | 光器件/光模块，日本光通信支柱 |

### 海外公司对 A 股的映射

| 海外动向 | A 股受益方向 | 具体标的 |
|---------|------------|---------|
| NVIDIA GB300/GB200 出货量超预期 | 光模块需求暴增 | 中际旭创、新易盛 |
| Lumentum EML 芯片扩产不及预期 | 光芯片涨价/国产替代加速 | 源杰科技、仕佳光子 |

| NVIDIA CPO 交换机交付 | 光引擎需求爆发 | 天孚通信 |
| Arista 交换机出货量指引上调 | 光模块配套需求增加 | 所有光模块标的 |
