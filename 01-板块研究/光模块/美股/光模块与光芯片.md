# 美股 — 光模块与光芯片公司分析

美股是全球光模块产业链的「上游核心」——光芯片（EML/CW/VCSEL）和电芯片（DSP）两大最核心环节被美国公司牢牢掌控。对 A 股投资者而言，跟踪这些公司可以预判光芯片供给瓶颈、技术路线演进和行业景气度。

---

## 一、光芯片与光模块巨头

### Lumentum（LITE）— EML 激光器全球龙头

| 维度 | 信息 |
|------|------|
| **定位** | 全球最大的 EML 激光器芯片和光模块供应商之一 |
| **核心产品** | EML 激光器（200G/通道）、VCSEL 芯片（Apple 3D 传感核心供应商）、CW 光源 |
| **NVIDIA 关联** | 据报道 NVIDIA 给 Lumentum 下了约 40 亿美金的长期订单，用于保障 800G/1.6T 光芯片供应 |
| **对 A 股的影响** | Lumentum 的 EML 扩产进度决定了全球光模块出货天花板。如果 Lumentum 扩产不及预期→光芯片紧缺→国产替代加速（源杰科技） |

### Coherent（COHR）— 光芯片+模块垂直整合

| 维度 | 信息 |
|------|------|
| **定位** | 全球第 3 光模块厂商 + 光芯片（VCSEL/CW/EML）+ 材料（SiC 衬底）平台型公司 |
| **光模块** | 800G/1.6T 光模块持续出货，全球 Top 3 |
| **光芯片** | CW 光源全球最大供应商之一，VCSEL 双寡头（与 Lumentum 分享 Apple 订单） |
| **对 A 股的意义** | Coherent 的财报是光模块行业最全面的景气度温度计——光模块营收 = 行业需求验证，光芯片营收 = 上游供给验证 |

---

## 二、DSP 电芯片双寡头

### Marvell（MRVL）— DSP 芯片绝对霸主

| 维度 | 信息 |
|------|------|
| **定位** | DSP 芯片全球第 1，市占率 ~60% |
| **光互联产品** | PAM4 DSP（800G/1.6T 光模块核心芯片）、定制 ASIC、数据中心交换机芯片 |
| **垄断地位** | 几乎所有高端光模块中的 DSP 芯片都是 Marvell 或博通——两家合计市占 >90% |
| **对 A 股的影响** | Marvell DSP 出货量 = 光模块行业需求的同步指标。DSP 供给紧张时→推动 LPO（去 DSP 化）加速→利好新易盛（首家 LPO 光模块量产厂商，LPO 独供英伟达） |

### Broadcom（AVGO）— 全栈光互联帝国

| 维度 | 信息 |
|------|------|
| **定位** | DSP #2 + 交换机芯片 #1 + 硅光芯片 + CPO 方案 |
| **产品矩阵** | Tomahawk 交换机芯片（51.2T/102.4T）、Jericho 路由芯片、PAM4 DSP、硅光引擎、CPO 交换机方案 |
| **独特优势** | 博通是全球唯一同时拥有「高性能交换机芯片 + DSP 芯片 + 硅光芯片」三大能力的公司——这使其在 CPO 时代拥有了最完整的方案整合能力 |
| **对 A 股的影响** | 博通的 Tomahawk 6 102.4T 交换机芯片量产节奏，决定了 1.6T 光模块的大规模需求何时到来 |

---

## 三、光互联新势力

### Credo（CRDO）— 高速 SerDes 专家

| 维度 | 信息 |
|------|------|
| **定位** | 高速 SerDes（串行解串器）IP 和有源电缆（AEC）龙头 |
| **核心产品** | 有源电缆（AEC，替代短距光模块/DAC 的方案）、高速 SerDes IP |
| **投资逻辑** | AEC 在短距（<7m）场景中替代光模块，成本更低、功耗更低。如果 AEC 渗透率提升→对短距光模块需求有一定替代效应 |
| **对 A 股的影响** | 关注 Credo 的 AEC 出货量——如果增速超预期，可能影响短距光模块的市场空间 |

### Arista Networks（ANET）— AI 交换机之王

| 维度 | 信息 |
|------|------|
| **定位** | 全球数据中心交换机龙头（超越思科），AI 集群组网核心 |
| **核心客户** | Meta、Microsoft、Oracle 等超大规模云厂商 |
| **AI 关联** | AI 集群需要大量高端交换机（51.2T/102.4T），Arista 是最大赢家 |
| **对 A 股的影响** | Arista 的交换机出货指引 = AI 集群建设速度的先行指标。出货量超预期→光模块需求超预期→所有 A 股光模块标的受益 |

---

## 四、NVIDIA：光模块的最大买家与 CPO 推动者

NVIDIA 不直接做光模块，但它是全球光模块最大的单一采购方，也是对光模块格局影响最大的公司：

| 角色 | 影响 |
|------|------|
| **最大买家** | GB200/GB300 出货量决定了 800G/1.6T 光模块的需求量 |
| **CPO 推动者** | Spectrum-X CPO 交换机 2026H2 交付，是首个商用 CPO 产品，决定 CPO 能否大规模落地 |
| **技术方向定义者** | NVIDIA 选择硅光还是传统 EML、支持 LPO 还是推 CPO，直接影响所有光模块厂商的技术路线选择 |
| **供应链分配者** | NVIDIA 给中际旭创多少份额、给新易盛多少份额，直接决定 A 股光模块龙头的业绩 |

> **跟踪 NVIDIA 对 A 股光模块的意义**：NVIDIA 的季度 GPU 出货指引 = 光模块需求的「锚」。NVIDIA 选择的光互联技术路线 = A 股光模块标的的未来方向。

## 美股各公司对 A 股的映射总结

| 美股公司 | 跟踪什么 | 映射 A 股标的 |
|---------|---------|-------------|
| **Lumentum** | EML 芯片产能/营收增速 | 源杰科技（国产替代） |
| **Coherent** | 光模块+光芯片双营收 | 中际旭创/新易盛 + 天孚通信 |
| **Marvell** | DSP 出货量 | LPO 加速 → 新易盛 |
| **Broadcom** | 102.4T 交换机芯片量产 | 1.6T 光模块需求 → 所有光模块标的 |
| **Credo** | AEC 出货量 | 短距光模块替代效应 |
| **Arista** | 交换机出货指引 | AI 集群需求 → 所有光模块标的 |
| **NVIDIA** | GPU 出货指引 + CPO 交付进度 | 中际旭创/天孚通信/所有光模块标的 |
